# moai-lifestyle

라이프스타일 플러그인 — 여행 플래닝, 건강/식단, 웨딩/이벤트.

## 스킬

| 스킬 | 설명 | 상태 |
|------|------|:----:|
| [travel-planner](./skills/travel-planner/) | 여행 플래너 — 일정, 맛집, 숙소, 예산 관리. 개인 재무, 부동산 분석, 사이드 프로젝트 런처 포함 | ✅ 완성 |
| [wellness-coach](./skills/wellness-coach/) | 웰니스 코치 — 운동 프로그램, 식단 플래너, 육아 가이드, 시니어 케어 | ✅ 완성 |
| [event-planner](./skills/event-planner/) | 이벤트 플래너 — 웨딩, 세미나, 워크샵, 컨퍼런스 기획 | ✅ 완성 |

## 사용 예시

```
5박 6일 오사카-교토 여행 계획 짜줘. 예산 80만원, 맛집 위주.
```

```
30대 직장인 다이어트 8주 식단 + 운동 플랜 만들어줘. 하루 1시간 운동 가능.
```
