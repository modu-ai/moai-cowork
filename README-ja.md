## 🎯 MoAIとは

**MoAI-Cowork**は、AIを組織内の各ドメイン専門家に変えるオート学習システムです。

- **100個のハーネス**: 10カテゴリ × 10ドメイン（会計、法務、HR、技術、マーケティングなど）
- **自己進化**: ユーザーフィードバックと結果分析によるSelf-Refineサイクル
- **グローバルプロファイル**: 個人、チーム、組織プロファイルに基づくカスタマイズ応答
- **プラグインベース**: Claude CodeとCoworkエコシステムとのシームレス統合

### 主要機能

| 機能 | 説明 |
|------|------|
| **自動学習** | 各インタラクションでのパフォーマンス追跡・分析 |
| **ドメイン専門性** | 各分野の最高実務知識の提供 |
| **文化適応** | グローバルビジネス慣行と多言語サポート |
| **リアルタイム更新** | 最新の税法、規制、市場情報を含む |
| **複数ユーザー対応** | チームレベルのプロファイル管理と協業 |

## 📦 インストール方法

### 要件
- Claude Code（最新版）
- Coworkプラグインサポート

### ステップ1: Coworkにプラグインを追加
```bash
# Cowork CLIを通じてインストール（予定）
cowork install moai-cowork
```

### ステップ2: プロファイル初期化
```bash
moai init
```

### ステップ3: 個人プロファイル設定
```bash
moai profile --set-personal
```

## 🚀 クイックスタート

### 最初のクエリを実行

```
@moai 2026年の日本の消費税申告期限は？
```

**応答**: MoAIは自動的に:
1. 日本のロケール情報をロード
2. 2026年の税法データを参照
3. カスタマイズされた消費税申告スケジュールを提供

### 自動学習を有効化

```
moai learn --feedback "答えが非常に正確でした"
```

MoAIはこのフィードバックを記録して、将来の同様なクエリを改善します。

## 🔄 自己学習（Self-Refine）サイクル

### 学習フロー

```
1. クエリ実行
   ↓
2. 応答生成（ハーネスベース）
   ↓
3. ユーザーフィードバック収集
   ↓
4. 結果分析（精度、関連性、有用性）
   ↓
5. ハーネス改善（重み付け調整）
   ↓
6. 次のクエリに適用
```

### フィードバックタイプ

| タイプ | 説明 | 影響 |
|--------|------|------|
| **肯定** | "非常に正確です" | ハーネス重み +10% |
| **部分** | "一部正確です" | 重み ±5% |
| **否定** | "間違っています" | ハーネス重み -15% |
| **カスタム** | "XX内容追加必要" | 特定分野強化 |

## 🛠 コントリビューション

### 1. 新しいハーネスを提案
```bash
# 新しいドメインを提案
moai contribute --domain "配送会社法" --category "labor"
```

### 2. 既存ハーネスを改善
```bash
# フィードバックベースの改善
moai improve DOMAIN_ID --feedback "内容追加が必要"
```

### 3. 新しいロケールを追加
```bash
# 新しい国のロケール追加
moai add-locale --country "オーストリア" --code "de_AT"
```

### 4. ドキュメントを改善
- `/skills/moai/references/locale/`ディレクトリのロケールファイルを編集
- GitHubでプルリクエストを提出
- 文化適応ガイドを更新
