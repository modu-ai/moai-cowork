## 🎯 MoAI क्या है?

**MoAI-Cowork** एक स्वचालित शिक्षण प्रणाली है जो AI को आपके संगठन के प्रत्येक क्षेत्र के लिए विशेषज्ञ में रूपांतरित करती है।

- **100 साधन**: 10 श्रेणियां × 10 डोमेन (लेखांकन, कानून, HR, प्रौद्योगिकी, विपणन, आदि)
- **आत्म-विकास**: उपयोगकर्ता प्रतिक्रिया और परिणाम विश्लेषण पर आधारित Self-Refine चक्र
- **वैश्विक प्रोफ़ाइल**: व्यक्तिगत, टीम और संगठनात्मक प्रोफ़ाइल के माध्यम से व्यक्तिगत प्रतिक्रियाएं
- **प्लगइन-आधारित**: Claude Code और Cowork पारिस्थितिकी तंत्र के साथ निरंतर एकीकरण

### मुख्य विशेषताएं

| कार्य | विवरण |
|------|--------|
| **स्वचालित शिक्षण** | प्रत्येक इंटरैक्शन पर प्रदर्शन रिकॉर्डिंग और विश्लेषण |
| **डोमेन विशेषज्ञता** | प्रत्येक क्षेत्र में अत्याधुनिक व्यावहारिक ज्ञान |
| **सांस्कृतिक अनुकूलन** | वैश्विक व्यावसायिक प्रथाओं और भाषाओं का समर्थन |
| **वास्तविक समय अद्यतन** | नवीनतम कर कानून, विनियम और बाजार की जानकारी शामिल |
| **बहु-उपयोगकर्ता समर्थन** | टीम स्तर की प्रोफ़ाइल प्रबंधन और सहयोग |

## 📦 स्थापना

### आवश्यकताएं
- Claude Code (नवीनतम संस्करण)
- Cowork प्लगइन समर्थन

### चरण 1: Cowork में प्लगइन जोड़ें
```bash
# Cowork CLI के माध्यम से स्थापना (जल्द ही)
cowork install moai-cowork
```

### चरण 2: प्रोफ़ाइल प्रारंभ करें
```bash
moai init
```

### चरण 3: व्यक्तिगत प्रोफ़ाइल कॉन्फ़िगर करें
```bash
moai profile --set-personal
```

## 🚀 त्वरित शुरुआत

### अपनी पहली क्वेरी चलाएं

```
@moai 2026 में भारत में GST फाइलिंग अनुसूची क्या है?
```

**प्रतिक्रिया**: MoAI स्वचालित रूप से:
1. भारत के स्थान की जानकारी लोड करता है
2. 2026 के कर कानून डेटा को संदर्भित करता है
3. व्यक्तिगत GST फाइलिंग अनुसूची प्रदान करता है

### स्वचालित शिक्षण सक्षम करें

```
moai learn --feedback "उत्तर बहुत सटीक था"
```

MoAI इस प्रतिक्रिया को भविष्य की समान क्वेरी को सुधारने के लिए रिकॉर्ड करता है।

## 📚 100 साधन कैटलॉग

### 10 श्रेणियां

#### 1️⃣ कर और लेखांकन (Tax & Accounting)
- **IN_TAX_001**: भारतीय GST और कर
- **US_TAX_001**: अमेरिकी संघीय आयकर
- **JP_TAX_001**: जापानी खपत कर
- **UK_TAX_001**: यूके VAT
- **VN_TAX_001**: वियतनामी VAT
- **TH_TAX_001**: थाई VAT
- *(6 अन्य)*

#### 2️⃣ श्रम कानून और HR (Labor & HR)
- **IN_HR_001**: भारतीय श्रम कानून
- **US_HR_001**: FLSA और न्यूनतम वेतन
- **JP_HR_001**: जापानी श्रम कानून
- **UK_HR_001**: यूके रोजगार कानून
- *(6 अन्य)*

#### 3️⃣ डेटा और अनुपालन (Data & Compliance)
- **IN_DATA_001**: व्यक्तिगत डेटा संरक्षण
- **US_DATA_001**: CCPA/HIPAA
- **JP_DATA_001**: व्यक्तिगत सूचना संरक्षण कानून
- **UK_DATA_001**: UK GDPR
- *(6 अन्य)*

#### 4️⃣ व्यावसायिक संचालन (Business Operations)
- **IN_BIZ_001**: भारतीय व्यावसायिक प्रथाएं
- **US_BIZ_001**: अमेरिकी बैठक संस्कृति
- **JP_BIZ_001**: जापानी सहयोग संस्कृति
- *(7 अन्य)*

#### 5️⃣ प्रौद्योगिकी और IT (Technology & IT)
- **TECH_001**: सॉफ्टवेयर विकास सर्वोत्तम प्रथाएं
- **TECH_002**: क्लाउड आर्किटेक्चर
- *(8 अन्य)*

#### 6️⃣ विपणन और बिक्रय (Marketing & Sales)
- **MKT_001**: डिजिटल विपणन रणनीति
- **MKT_002**: B2B बिक्रय तकनीकें
- *(8 अन्य)*

#### 7️⃣ वित्त और निवेश (Finance & Investment)
- **FIN_001**: वित्तीय विवरण विश्लेषण
- **FIN_002**: निवेश पोर्टफोलियो प्रबंधन
- *(8 अन्य)*

#### 8️⃣ कानून और अनुबंध (Legal & Contracts)
- **LEG_001**: अनुबंध समीक्षा
- **LEG_002**: NDA ड्राफ्टिंग
- *(8 अन्य)*

#### 9️⃣ रणनीति और योजना (Strategy & Planning)
- **STR_001**: व्यावसायिक रणनीति स्थापना
- **STR_002**: OKR सेटिंग
- *(8 अन्य)*

#### 🔟 ग्राहक और सेवा (Customer & Service)
- **CUS_001**: ग्राहक संतुष्टि विश्लेषण
- **CUS_002**: सेवा सुधार योजना

## 🔄 आत्म-शिक्षण चक्र (Self-Refine)

### शिक्षण प्रवाह

```
1. क्वेरी चलाएं
   ↓
2. प्रतिक्रिया उत्पन्न करें (साधन का उपयोग करते हुए)
   ↓
3. उपयोगकर्ता प्रतिक्रिया एकत्र करें
   ↓
4. परिणाम विश्लेषण करें (सटीकता, प्रासंगिकता, उपयोगिता)
   ↓
5. साधन सुधारें (वजन समायोजित करें)
   ↓
6. अगली क्वेरी पर लागू करें
```

### प्रतिक्रिया प्रकार

| प्रकार | विवरण | प्रभाव |
|--------|--------|--------|
| **सकारात्मक** | "बहुत सटीक" | साधन वजन +10% |
| **आंशिक** | "आंशिक रूप से सही" | वजन ±5% |
| **नकारात्मक** | "गलत" | साधन वजन -15% |
| **कस्टम** | "X के बारे में अधिक सामग्री की आवश्यकता" | विशिष्ट क्षेत्र सुदृढ़ीकरण |

## 📖 उपयोग के उदाहरण

### उदाहरण 1: कर सलाह
```
प्रश्न: "भारत में नए कर्मचारी के लिए EPF पंजीकरण प्रक्रिया क्या है?"
→ साधन IN_HR_001 सक्रिय करें
→ समय सीमा, आवश्यक दस्तावेज और प्रक्रिया प्रदान करें
```

### उदाहरण 2: अंतर्राष्ट्रीय व्यावसायिक
```
प्रश्न: "अमेरिकी व्यावसायिक बैठक में मुझे क्या ध्यान रखना चाहिए?"
→ साधन US_BIZ_001 सक्रिय करें
→ सांस्कृतिक अनुकूलन मार्गदर्शन, समय प्रबंधन, संचार शैली
```

### उदाहरण 3: नियामक अनुपालन
```
प्रश्न: "EU में ग्राहक डेटा प्रोसेस करने के लिए मुझे कौन सी प्रक्रियाएं करनी हैं?"
→ साधन IN_DATA_001 और संबंधित GDPR सक्रिय करें
→ GDPR अनुपालन, सहमति प्रबंधन, डेटा स्थानांतरण प्रक्रिया
```

## 🛠 योगदान कैसे करें (Contributing)

### 1. नया साधन प्रस्तावित करें
```bash
# नया डोमेन प्रस्तावित करें
moai contribute --domain "भारतीय वाणिज्यिक कानून" --category "legal"
```

### 2. मौजूदा साधन सुधारें
```bash
# प्रतिक्रिया-आधारित सुधार
moai improve DOMAIN_ID --feedback "अतिरिक्त सामग्री आवश्यक"
```

### 3. नया स्थानीयकरण जोड़ें
```bash
# नया देश स्थानीयकरण जोड़ें
moai add-locale --country "श्रीलंका" --code "si_LK"
```

### 4. दस्तावेज़ सुधारें
- `/skills/moai/references/locale/` में स्थानीयकरण फ़ाइलें संपादित करें
- GitHub पर Pull Request सबमिट करें
