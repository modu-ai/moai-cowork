# moai-support

고객 지원 플러그인 — 티켓 분류, 응답 초안, KB 문서, 에스컬레이션 관리.

## 스킬

| 스킬 | 설명 | 상태 |
|------|------|:----:|
| [ticket-triage](./skills/ticket-triage/) | 티켓 분류 — 고객 지원 티켓 우선순위 분류, CS 응대 가이드, 에스컬레이션 프로세스 | 📝 초안 |
| [draft-response](./skills/draft-response/) | 응답 초안 — 고객 문의 이메일 답변, 채팅 응대 메시지, 공식 답변서 초안 생성 | 📝 초안 |
| [kb-article](./skills/kb-article/) | KB 문서 작성 — FAQ, 사용자 가이드, 트러블슈팅, 헬프 센터 아티클 | 📝 초안 |
| [escalation-manager](./skills/escalation-manager/) | 에스컬레이션 관리 — 고객 불만 처리, 피드백 분석, 설문 설계, VIP 응대 | 📝 초안 |

## 사용 예시

```
환불 요청 고객에게 보낼 이메일 초안 써줘. 정책상 7일 이내만 환불 가능해.
```

```
배송 관련 자주 묻는 질문 FAQ 10개 작성해줘
```
