# 면접 코치 (interview-coach)

> moai-career v1.0.0

## 지원 영역

| 영역 | 설명 |
|------|------|
| 예상 질문 생성 | 직무별, 기업별, 경력별 맞춤 면접 질문 리스트 |
| 모의 면접 | 질문 → 답변 → 피드백 → 개선 답변 루프 |
| 유형별 대비 | AI면접, 역량면접(BEI), PT면접, 토론면접, 임원면접 |
| 답변 코칭 | STAR 기반 답변 구조화, 시간 배분, 비언어 커뮤니케이션 |

참조 가이드: `references/common-questions.md`, `references/interview-types.md`

## 면접 유형별 대비 전략

### AI 면접 (AI 역량검사)

2026년 기준 대기업 320개 이상이 도입. 기아, 농심, 롯데, LG유플러스, SK하이닉스, KT, KB국민은행 등이 활용합니다.

| 단계 | 내용 | 대비 포인트 |
|------|------|------------|
| 자기소개 | 1분 영상 녹화 | 카메라 응시, 밝은 표정, 핵심 3줄 |
| 기본 질문 | 지원동기, 강점/약점 | 30초 내 두괄식 답변 |
| 성향 파악 | 상황 판단 선택지 | 일관성 유지, 극단 회피 |
| 상황 대처 | 갈등/실패 경험 | STAR 기법 + 감정 표현 |
| 전략 게임 | 인지력/판단력 테스트 | 연습 게임으로 패턴 익히기 |
| 심층 대화 | AI 꼬리질문 | 구체적 수치와 사례 준비 |

**핵심**: AI는 답변 텍스트 + 표정 + 음성을 종합 분석합니다. 자연스러운 표정 관리와 일정한 발화 속도가 중요합니다.

### 역량 면접 (BEI: Behavioral Event Interview)

과거 행동에서 미래 성과를 예측하는 구조화 면접입니다.

- **질문 패턴**: "~했던 경험을 말해주세요", "어떻게 해결했나요?"
- **답변 공식**: STAR(상황-과제-행동-결과) + 배운 점
- **평가 기준**: 구체성, 본인 역할 명확성, 결과의 정량화

### PT 면접

주어진 주제(30~60분 준비)에 대해 10~15분 발표 후 Q&A를 진행합니다.

| 항목 | 전략 |
|------|------|
| 구조 | 문제 정의 → 원인 분석 → 해결 방안 → 기대 효과 |
| 슬라이드 | 5~7장, 데이터/도표 활용, 텍스트 최소화 |
| 시간 배분 | 준비 시간의 50% 구조 설계, 30% 자료 작성, 20% 리허설 |
| Q&A 대비 | "왜 그 방안인가?", "리스크는?", "대안은?" 예상 |

### 토론 면접

4~6인 그룹으로 찬반/자유 토론을 진행합니다.

- **역할 선택**: 사회자(리더십), 타임키퍼(책임감), 발표자(표현력) 중 선택
- **핵심**: 논리적 주장 + 경청 + 건설적 반론. 공격적 반박은 감점 요소

### 임원 면접

최종 관문으로 인성, 조직 적합성, 비전을 평가합니다.

- **빈출 질문**: "왜 이 회사인가?", "5년 후 목표", "우리 회사의 도전 과제"
- **핵심**: 기업 비전과 개인 목표의 연결, 진정성 있는 태도

## 모의 면접 루프

```
[1] 면접관 역할로 질문 제시
    ↓
[2] 사용자 답변 입력
    ↓
[3] 피드백 제공 (구조/내용/전달력 3축 평가)
    ↓
[4] 개선된 모범 답변 제시
    ↓
[5] 다음 질문으로 이동 또는 같은 질문 재시도
```

피드백 기준:
- **구조** (30%): STAR 기법 준수, 논리적 흐름
- **내용** (50%): 구체성, 직무 연관성, 차별화 포인트
- **전달력** (20%): 두괄식 여부, 적절한 길이(1~2분)

## 트리거 키워드

면접, 인터뷰, 모의면접, 예상질문, 면접준비, AI면접, 역량면접, PT면접, 토론면접, 임원면접

## 사용 예시

- "카카오 백엔드 개발자 면접 예상 질문 10개 뽑아줘"
- "모의 면접 해줘. 마케팅 직무, 대기업 1차 실무면접"
- "AI면접 대비법 알려줘. SK하이닉스 지원 예정"
- "PT면접 주제 '디지털 전환 전략' 발표 구조 잡아줘"
- "내 답변 피드백해줘: '저는 팀 프로젝트에서 리더 역할을...'"

## 독립 실행 워크플로우

참조 가이드(`references/`)를 사용할 수 없는 경우 다음 단계로 실행합니다.

### 1단계: 면접 컨텍스트 파악
- 면접 유형: AI면접/역량/PT/토론/임원
- 기업 정보: 기업명, 직무, 면접 차수
- 사용자 배경: 전공, 경력, 주요 경험

### 2단계: 맞춤 질문 생성
- 직무 역량 질문 (60%): 기술적 지식, 실무 경험
- 조직 적합성 질문 (30%): 협업, 갈등 해결, 가치관
- 성장 의지 질문 (10%): 학습 계획, 커리어 비전

### 3단계: 모의 면접 또는 답변 코칭
- 모의 면접: 질문-답변-피드백 루프 반복
- 답변 코칭: 사용자 답변을 STAR 구조로 재구성

### 4단계: 종합 피드백
- 강점/약점 요약
- 개선 우선순위 제시
- 추가 연습 추천 질문 리스트

## 실행 규칙

1. 사용자 요청 수신 → 면접 유형 및 기업 정보 파악
2. `references/{id}.md` 존재 시 로드 → 가이드에 따라 실행
3. `--deepthink` 또는 복잡한 기업 분석 기반 면접 → `mcp__sequential-thinking__sequentialthinking` 호출
4. 모의 면접 시 1문제씩 순차 진행, 사용자 답변 대기

## 문제 해결

- **기업별 면접 정보 부족**: 기업명을 알려주면 공개된 면접 후기와 기출 패턴을 기반으로 질문을 생성합니다. 잡플래닛, 블라인드 후기를 참고하면 더 정확합니다.
- **답변이 너무 짧거나 길 때**: 실무면접 답변은 1~2분(300~500자), AI면접은 30초~1분이 적절합니다. 타이머를 활용하세요.
- **긴장 관리**: 모의 면접을 5회 이상 반복하면 실전 긴장이 크게 줄어듭니다. 녹음/녹화 후 자기 모니터링을 병행하세요.
- **영어 면접 대비**: 외국계 기업이나 영어 면접이 필요하면 영어 답변 모드로 전환할 수 있습니다.

## 공유 에이전트

이 플러그인에서 활용할 수 있는 다른 플러그인의 에이전트:

| 에이전트 | 소속 | 용도 |
|---------|------|------|
| quality-evaluator | moai-core | 산출물 품질 PASS/FAIL 판정 |
| korean-tone-reviewer | moai-hr | 직급별 경어 사용 및 비즈니스 톤 적절성 검토 |

## 이 스킬을 사용하지 말아야 할 때

- **자기소개서 작성**: 자소서/이력서 작성은 `resume-builder` 스킬을 사용하세요.
- **채용공고 분석**: JD 분석이나 기업 리서치는 `job-analyzer` 스킬이 적합합니다.
- **채용 담당자용 면접 설계**: 기업 인사 담당자가 면접을 설계하려면 moai-hr의 `employment-manager` 스킬을 사용하세요.
- **코딩 테스트 준비**: 알고리즘/자료구조 문제 풀이는 이 스킬의 범위가 아닙니다.
