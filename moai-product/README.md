# moai-product

제품 혁신 플러그인 — PM 로드맵, UX 리서치, 스프린트 플래닝, 제품 스펙.

## 스킬

| 스킬 | 설명 | 상태 |
|------|------|:----:|
| [spec-writer](./skills/spec-writer/) | 스펙 작성 — PRD, 기능 명세서, AI/ML 전략, 정부 지원금(R&D, SBIR) 기획 | ✅ 완성 |
| [roadmap-manager](./skills/roadmap-manager/) | 로드맵 관리 — 프로젝트 트래커, 마일스톤, 파트너십 개발, MOU, 전략적 제휴 | ✅ 완성 |
| [ux-researcher](./skills/ux-researcher/) | UX 리서처 — 사용자 인터뷰, 유저빌리티 테스트, 페르소나, VOC, NPS 분석 | ✅ 완성 |

## 사용 예시

```
신규 기능 PRD 작성해줘. AI 기반 추천 시스템, 모바일 앱, 3개월 개발 일정.
```

```
우리 앱 사용자 인터뷰 가이드 만들어줘. 이탈률이 높은 온보딩 화면 개선이 목적이야.
```
