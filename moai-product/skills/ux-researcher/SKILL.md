## 2026 UX 리서치 트렌드 컨텍스트

한국 PM/UX 커뮤니티 및 글로벌 CX 트렌드 반영:

### 2026 한국 UX 리서치 방법론

| 방법 | 설명 | 한국 특화 고려사항 |
|------|------|-------------------|
| 사용자 인터뷰 | 1:1 심층 인터뷰 30~60분 | 유교적 문화로 직접 비판 회피. 간접 표현 분석 필요 |
| 유저빌리티 테스트 | 실제 태스크 수행 관찰 | 한국어 UI 가독성, 모바일 First |
| 설문 조사 | 정량 데이터 수집 | 카카오폼, 네이버폼, Google Forms 활용 |
| 다이어리 스터디 | 2~4주 사용 일기 | 카카오톡 채널을 통한 일일 체크인 |
| A/B 테스트 | 변형 버전 비교 | 네이버/카카오 SDK의 통계 유의성 도구 활용 |
| VOC 분석 | 고객 리뷰·CS 데이터 분석 | 네이버 리뷰, 카카오 채널 메시지, 앱스토어 리뷰 |

### AI 기반 UX 리서치 (2026)

- **자동 인터뷰 분석**: AI로 인터뷰 녹취 → 텍스트 변환 → 주제 클러스터링
- **감성 분석**: 고객 리뷰·SNS 멘션의 긍정/부정/중립 자동 분류
- **페르소나 자동 생성**: 사용자 데이터 기반 AI 페르소나 초안 생성
- **히트맵 분석**: Hotjar, Microsoft Clarity 기반 클릭/스크롤 패턴 분석

### 한국 서비스 UX 특성

- **모바일 퍼스트**: 한국 인터넷 사용의 80%+ 이상이 모바일
- **카카오 생태계**: 카카오 로그인·결제·채널과의 UX 일관성 중요
- **빠른 응답 기대**: 페이지 로드 2초 초과 시 이탈률 급상승 (한국 사용자 특성)
- **고령 친화**: 폰트 크기, 버튼 크기, 명도 대비 접근성 고려 증가
