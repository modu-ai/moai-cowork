# 데일리 브리핑 (Daily Briefing)

> moai-business v1.0.0

## 개요

매일 아침 업무 시작 전 5분 만에 비즈니스 현황을 파악할 수 있는 구조화된 브리핑을 생성한다.
국내외 업계 뉴스, 경쟁사 동향, 주요 지표, 오늘의 액션 아이템을 하나의 리포트로 통합한다.
사용자 업종·관심 주제에 맞게 브리핑 항목을 맞춤 설정할 수 있다.

## 트리거 키워드

일일 브리핑, 데일리 리포트, 오늘의 이슈, 오늘 뉴스, 아침 브리핑, 비즈니스 동향, 시장 동향, 뉴스 요약, 경쟁사 모니터링, 주간 브리핑, 위클리 리포트

## 워크플로우

### 1단계: 브리핑 범위 설정
- 대상 업종 및 키워드 확인 (`.moai/config.json`의 `industry` 설정 우선 사용)
- 브리핑 기간 설정 (일간/주간/월간)
- 모니터링 경쟁사 목록 확인

### 2단계: 정보 수집 및 분류
다음 섹션으로 구조화하여 수집:
- **국내 업계 뉴스**: 네이버 뉴스, 조선일보 경제, 한국경제, 파이낸셜뉴스
- **글로벌 동향**: WSJ, TechCrunch, Bloomberg 주요 헤드라인 요약
- **경쟁사 모니터링**: 경쟁사 채용 공고, 보도자료, 제품 업데이트
- **규제·정책 변화**: 금융위, 공정거래위, 중기부, 과기정통부 공지
- **시장 지표**: 코스피/코스닥, 환율, 유가, 금리 동향

### 3단계: 브리핑 리포트 생성
아래 표준 포맷으로 마크다운 리포트 생성:

```
# [날짜] 데일리 브리핑

## 헤드라인 요약 (3줄 핵심)
1. ...
2. ...
3. ...

## 업계 주요 뉴스
| 제목 | 출처 | 시사점 |

## 경쟁사 동향

## 규제/정책 업데이트

## 오늘의 액션 아이템
- [ ] ...
```

### 4단계: 사용자 검토 및 저장
- 리포트를 사용자에게 제시
- 피드백 반영 후 `.moai/briefings/YYYYMMDD.md`에 저장 (옵션)

## 사용 예시

**예시 1**: "오늘 SaaS 업계 브리핑 해줘"
→ SaaS 관련 국내외 뉴스 수집 → 경쟁사 동향 → 규제 변화 → 액션 아이템 리포트 생성

**예시 2**: "이번 주 핀테크 시장 동향 요약하고 경쟁사 카카오페이, 토스 최근 동향 포함해줘"
→ 주간 핀테크 뉴스 → 지정 경쟁사 별도 섹션 → 시사점 및 대응 방안

## 산출물

- 마크다운 형식의 구조화된 일일 브리핑 리포트
- 주요 뉴스 출처 링크 포함
- 경쟁사 모니터링 요약
- 오늘의 액션 아이템 체크리스트

## 참고 사항

### 한국 주요 정보 소스
- **경제/비즈니스**: 한국경제(hankyung.com), 매일경제(mk.co.kr), 파이낸셜뉴스(fnnews.com)
- **스타트업/테크**: 블로터(bloter.net), 벤처스퀘어(venturesquare.net), 이코노미스트
- **정책/규제**: 정책브리핑(korea.kr), 중기부 공식 채널
- **금융 지표**: 네이버 금융, KRX 한국거래소, 한국은행 경제통계시스템(ECOS)

### 브리핑 자동화 팁
- `.moai/config.json`에 `industry`, `competitors`, `keywords` 설정 시 맞춤 브리핑 자동화
- 사용자가 "매일 아침 브리핑" 요청 시 이전 브리핑과 비교하여 변화 포인트를 강조
- TAM·SAM·SOM 기반 시장 동향 분석 가능 (moai-business/market-analyst 스킬과 연동)

### TAM·SAM·SOM 시장 규모 분석 (투자자 브리핑 모드)
사업계획서 또는 IR 브리핑 요청 시 다음 공공 데이터 소스 활용:
- 통계청 KOSIS(kosis.kr)
- 한국무역협회 K-stat(stat.kita.net)
- 중소벤처기업부 K-스타트업(k-startup.go.kr) 통계

## 문제 해결

| 상황 | 대응 |
|------|------|
| 실시간 뉴스 접근이 어려운 경우 | 알려진 최신 동향을 기반으로 브리핑 구성. 실시간 검색 권고 |
| 업종 키워드가 불명확한 경우 | 사용자에게 주요 키워드 2~3개 확인 후 진행 |
| 경쟁사 정보가 부족한 경우 | 공개 자료(IR, 언론 보도, 채용 공고)로 추론하여 제공 |

## 이 스킬을 사용하지 말아야 할 때

- 심층 시장 분석이나 TAM/SAM/SOM 산출 → moai-business/market-analyst 사용
- 투자자 보고서나 IR 덱 작성 → moai-business/investor-relations 사용
- 마케팅 성과 분석 보고서 → moai-marketing/performance-report 사용
- 단순 뉴스 검색이나 일반 질문 → 일반 대화로 해결
