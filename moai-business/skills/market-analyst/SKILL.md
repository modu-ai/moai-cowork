# 시장 분석가 (Market Analyst)

> moai-business | 시장 조사 및 경쟁 분석 전문 스킬

## 하네스 (전략 가이드)

| ID | 한국명 | 설명 |
|----|--------|------|
| market-research | 시장 조사 | TAM/SAM/SOM, 시장 트렌드, 고객 세그멘테이션 |
| competitive-analysis | 경쟁 분석 | 경쟁사 매핑, 차별화 전략, 포지셔닝 |
| pricing-strategy | 가격 전략 | 가격 모델, 탄력성 분석, 번들링 |

→ 하네스 파일: `references/{id}.md`

### 기본 실행 워크플로우 (references 미로드 시)

1. 사용자 요청에서 목표/산출물/대상 추출
2. 해당 도메인의 표준 프레임워크 적용 (TAM/SAM/SOM, 포지셔닝 매트릭스, 가격 탄력성)
3. 초안 생성 → 사용자 검토 요청
4. 피드백 반영하여 최종본 완성

## 트리거 키워드

시장조사, TAM, SAM, SOM, 경쟁분석, 경쟁사, 포지셔닝, 차별화, 가격전략, 가격 모델, 고객 세그멘테이션

## 워크플로우

### 1단계: 분석 범위 확인
- 목표 시장 및 지역 범위 확인 (국내 / 글로벌)
- 분석 유형 선택 (시장 규모 / 경쟁사 / 가격 / 전체)
- 보유 데이터 및 추가 필요 정보 파악

### 2단계: 시장 규모 분석 (TAM/SAM/SOM)
- **TAM** (전체 시장): 업계 전체 잠재 매출 규모
- **SAM** (서비스 가능 시장): 실제 접근 가능한 세그먼트
- **SOM** (현실적 목표 시장): 3~5년 내 점유 가능한 규모
- 근거 데이터: 통계청, 업계 리포트, 유사 기업 IR 자료

### 3단계: 경쟁사 분석
- 직접·간접 경쟁사 식별 및 분류
- 포지셔닝 매트릭스 (가격 vs 품질, 기능 vs 편의성 등)
- 경쟁사 강점·약점 비교 테이블
- 차별화 기회 영역 도출

### 4단계: 가격 전략 (선택)
- 원가 기반 / 경쟁사 기준 / 가치 기반 가격 모델 비교
- 가격 탄력성 분석
- 번들링·구독·프리미엄 등 가격 구조 설계

### 5단계: 보고서 생성
- 시장 분석 보고서 (요약 + 상세)
- 인사이트 및 전략 권고안

## 사용 예시

**예시 1**: "B2B SaaS 인사관리 솔루션 국내 시장 규모 분석해줘"
→ TAM(전체 HR SW 시장) → SAM(SME 타겟) → SOM(3년 목표) → 경쟁사 매핑

**예시 2**: "경쟁사 3곳 가격 전략 비교하고 우리 가격 어떻게 설정할지 조언해줘"
→ 경쟁사 가격 조사 → 포지셔닝 분석 → 가격 모델 제안

## 실행 규칙

1. 사용자 요청 수신 → 해당 하네스 판별
2. `references/{id}.md` 로드 → 분석 가이드에 따라 실행
3. `--deepthink` 또는 복잡 분석 → `mcp__sequential-thinking__sequentialthinking` 호출
4. 결과물 생성 후 사용자 검토 요청

## 문제 해결

| 상황 | 대응 |
|------|------|
| 공개 시장 데이터가 부족한 경우 | 유사 해외 시장 데이터 참조 + 국내 비율 추정. 근거와 한계 명시 |
| 경쟁사 정보가 제한적인 경우 | 공개 자료(IR, 뉴스, 채용 공고)로 추론. 추정임을 명시 |
| 분석 범위가 너무 넓은 경우 | 핵심 세그먼트 1~2개로 집중하여 분석 시작 |

## 이 스킬을 사용하지 말아야 할 때

- 사업계획서, 비즈니스 모델 캔버스, SWOT → moai-business/strategy-planner 사용
- 투자자 IR 자료나 재무 모델링 → moai-business/investor-relations 사용
- 마케팅 캠페인 성과 분석 → moai-marketing/performance-report 사용
- 단순 정보 검색이나 일반 질문 → 일반 대화로 해결
