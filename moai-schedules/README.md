# moai-schedules

스케줄 관리 플러그인 — 반복 업무 자동 실행, 캘린더 등록, 일정 조회/관리.

다른 `moai-*` 스킬을 정해진 시간에 자동으로 실행하여 반복 업무를 자동화합니다.

## 스킬

| 스킬 | 설명 | 상태 |
|------|------|:----:|
| [create-schedule](./skills/create-schedule/) | 일정 생성 — 새 미팅, 반복 일정 설정, 참석자 초대, 알림 구성, 캘린더 등록 | 📝 초안 |
| [list-schedules](./skills/list-schedules/) | 일정 조회 — 기간별 일정 목록, 필터링, 오늘/이번 주/이번 달 캘린더 뷰 | 📝 초안 |
| [manage-schedule](./skills/manage-schedule/) | 일정 관리 — 기존 일정 수정, 취소, 재예약, 반복 일정 패턴 관리 | 📝 초안 |

## 사용 예시

```
매주 월요일 오전 9시에 weekly report 자동 생성해줘
```

```
이번 주 일정 목록 보여줘
```
