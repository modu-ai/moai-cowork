# 스케줄 목록 조회 (List Schedules)

> moai-schedules v1.0.0

## 개요

Claude Cowork에 등록된 자동화 스케줄 목록과 실행 상태를 조회합니다. 각 스케줄의 마지막 실행 결과, 다음 실행 예정 시간, 건너뛴 실행 이력을 확인하고 현재 활성/비활성 상태를 파악합니다.

## 트리거 키워드

스케줄 조회, 자동화 목록, 예약 태스크 확인, 실행 이력, 스케줄 현황, 등록된 스케줄, scheduled tasks 보기, 자동화 현황

## 워크플로우

1. **조회 범위 선택**: 전체 / 활성 스케줄만 / 특정 스케줄
2. **상태 확인**:
   - 활성(Active): 정상 작동 중
   - 일시정지(Paused): 사용자가 일시 중단
   - 오류(Error): 마지막 실행 실패
3. **실행 이력 표시**: 최근 N회 실행 결과 (성공/실패/건너뜀)
4. **다음 실행 시간**: 현재 시각 기준 남은 시간 계산
5. **요약 리포트**: 전체 스케줄 현황 마크다운 표 생성

## Cowork에서 스케줄 조회하는 방법

### 방법 1: Scheduled Tasks 페이지
1. Claude Desktop 사이드바에서 **"Scheduled"** 클릭
2. 등록된 모든 스케줄 목록 확인
3. 각 스케줄 클릭 → 상세 정보 및 실행 이력 확인

### 방법 2: 이 스킬을 통한 요약 조회
현재 Cowork에 등록된 스케줄 목록을 자연어로 요약하거나, 스케줄 계획을 정리하는 데 활용합니다.

## 스케줄 상태 대시보드 형식

```markdown
## Cowork 스케줄 현황 (기준: YYYY-MM-DD HH:MM)

| 스케줄 이름 | 주기 | 상태 | 마지막 실행 | 다음 실행 | 연결 스킬 |
|------------|------|------|------------|----------|-----------|
| 매일 아침 브리핑 | 매일 8:30 | ✅ 활성 | 오늘 8:30 성공 | 내일 8:30 | daily-briefing |
| 주간 마케팅 리포트 | 매주 월 9:00 | ✅ 활성 | 월 09:00 성공 | 다음 주 월 | performance-report |
| 월간 세무 체크 | 매월 20일 | ⏸ 일시정지 | 3/20 성공 | — | tax-helper |
| 주간 CS 요약 | 매주 금 17:00 | ✅ 활성 | 금 17:00 성공 | 이번 주 금 | escalation-manager |

### 요약
- 전체 스케줄: 5개
- 활성: 4개 | 일시정지: 1개 | 오류: 0개
- 오늘 예정 실행: 1개
```

## 실행 이력 조회 형식

```markdown
## [스케줄 이름] 실행 이력

| 실행 일시 | 상태 | 소요 시간 | 비고 |
|----------|------|----------|------|
| 2026-04-09 08:30 | ✅ 성공 | 45초 | — |
| 2026-04-08 08:30 | ✅ 성공 | 38초 | — |
| 2026-04-07 08:30 | ⏭ 건너뜀 | — | 앱 종료 중 |
| 2026-04-07 08:35 | ✅ 재실행 | 42초 | 앱 재시작 후 자동 실행 |
```

## 건너뜀(Skip) 처리 안내

Cowork 공식 동작:
- 컴퓨터 절전/앱 종료 중 예정 시간 도달 시 → 스케줄 건너뜀
- 앱 재시작 또는 컴퓨터 켜질 때 → 건너뛴 작업 자동 재실행
- 재실행 시 앱 상단에 알림 표시
- 실행 이력에 "건너뜀 → 재실행" 이력 기록

## 산출물

- 전체 스케줄 현황 마크다운 표
- 각 스케줄별 실행 이력 요약
- 상태 이상(오류/건너뜀 반복) 스케줄 알림
- 최적화 권고 (불필요한 스케줄, 주기 조정 제안)

## 참고 사항

- **실시간 조회**: Cowork UI에서만 실시간 상태 확인 가능. 이 스킬은 사용자가 제공한 정보 기반으로 요약 작성
- **이력 보존**: Cowork는 최근 실행 이력을 태스크별로 보관 (정확한 보존 기간은 플랜별 상이)
- **공유 불가**: 스케줄 설정은 계정 단위. Team 플랜에서는 조직 내 공유 기능 확인 필요
