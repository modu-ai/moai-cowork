## 트리거 키워드

리서치, 논문, 학술 논문, 데이터 수집, 데이터 분석, 보고서, 문헌 검토, 피어 리뷰, 연구 설계

## 실행 규칙

1. 사용자 요청 수신 → 해당 전략 가이드 판별
2. `references/{id}.md` 로드 → 전략 가이드에 따라 실행
3. `--deepthink` 또는 복잡 학술 분석 → `mcp__sequential-thinking__sequentialthinking` 호출
4. 결과물 생성 후 사용자 검토 요청

## 사용 예시

- "AI 교육 효과에 관한 문헌 검토 보고서를 작성해줘"
- "설문 데이터를 분석해서 인사이트를 도출해줘"
- "석사 논문 연구 계획서 구조를 잡아줘"
- "APA 형식으로 참고문헌을 정리해줘"
- "이 논문의 연구 방법론을 검토하고 피드백 줘"

## 독립 실행 워크플로우

전략 가이드 파일 없이도 다음 단계로 실행 가능합니다:

1. **연구 주제 확인**: 연구 목적, 연구 질문(RQ), 대상 독자 및 발표 지면 파악
2. **문헌 조사 계획**: 국내(RISS, DBpia) 및 해외(Google Scholar, PubMed) 검색 전략 수립
3. **자료 분석**: 수집된 문헌·데이터 분류, 주제 클러스터링, 핵심 논점 도출
4. **논문 구조 설계**: 서론→선행연구→연구방법→결과→논의→결론 구조로 개요 작성
5. **초안 작성 및 검토**: 각 섹션 초안 작성, 논리 흐름 점검, 인용 형식 확인

### 에듀테크·AI 교육 트렌드 (2026)

DBpia·RISS 최신 논문 동향 반영:

- **AI 리터러시 연구**: 언어교육 전공 학생의 AI 인식 및 역량 탐구 논문 증가
- **학습자 모델링**: AI 코스웨어 기반 개인 맞춤형 학습 경로 설계 연구 활발
- **AI 윤리**: 빅데이터 기반 에듀테크의 윤리적 사용에 대한 담론 증가
- **AI 코스웨어**: 디지털 교과서 + AI 접목, 자동채점·오답 클리닉·학습 분석 통합
- **초개인화 학습**: PwC 삼일회계법인 보고서 — 에듀테크의 핵심은 '1인 1맞춤 학습'

### Bloom's Taxonomy 활용

에듀테크·AI 교육 설계에서 블룸의 교육목표분류학 적용:

| 인지 수준 | 설명 | AI 활용 방식 |
|-----------|------|-------------|
| 기억(Remember) | 사실 암기 | 플래시카드, 자동 퀴즈 |
| 이해(Understand) | 개념 설명 | 챗봇 Q&A, 요약 생성 |
| 적용(Apply) | 실제 사용 | 시뮬레이션, 실습 과제 |
| 분석(Analyze) | 구조 파악 | 데이터 분석, 비교 |
| 평가(Evaluate) | 판단·비판 | AI 피드백, 피어 리뷰 |
| 창조(Create) | 새로운 산출 | 프로젝트, 포트폴리오 |

→ AI는 하위 인지 수준(기억~적용)을 자동화하고, 교사는 상위 수준(분석~창조) 지도에 집중하는 구조 권장

## 문제 해결

| 상황 | 해결 방법 |
|------|-----------|
| 교육 자료 접근 불가 | 접근 가능한 자료 범위를 알려주시면 그 안에서 최대한 분석해 드립니다 |
| 학습 목표 불명확 | 연구 질문(RQ)을 함께 구체화하는 과정부터 시작해 드립니다 |
| 원시 데이터 없음 | 기존 선행 연구 메타분석 또는 문헌 고찰 방향으로 전환할 수 있습니다 |
| 저널별 투고 규정 상이 | 대상 저널명을 알려주시면 해당 투고 규정에 맞춰 조정합니다 |
