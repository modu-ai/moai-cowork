## 트리거 키워드

시험, 기출, 자격증, 수능, 문제 출제, 평가, 퀴즈, 학습 평가, 모의고사, TOEIC, TOEFL

## 실행 규칙

1. 사용자 요청 수신 → 시험/평가 유형 판별
2. `references/exam-prep.md` 로드 → 가이드에 따라 실행
3. `--deepthink` 또는 복잡 분석 → `mcp__sequential-thinking__sequentialthinking` 호출
4. 결과물 생성 후 사용자 검토 요청

## 사용 예시

- "정보처리기사 실기 모의문제 10개를 만들어줘"
- "이 단원에서 나올 수 있는 내신 예상 문제를 출제해줘"
- "TOEIC 700점 목표 4주 학습 계획을 세워줘"
- "공무원 시험 행정학 핵심 개념 퀴즈를 만들어줘"
- "수강생 이해도를 평가할 형성평가 문항을 설계해줘"

## 독립 실행 워크플로우

전략 가이드 파일 없이도 다음 단계로 실행 가능합니다:

1. **평가 목적 파악**: 형성평가(학습 중 확인) vs 총괄평가(학습 후 측정), 대상 시험 종류 확인
2. **출제 범위 설정**: 학습 목표, 단원, 난이도(상/중/하) 비율 결정
3. **문항 유형 선택**: 선택형(4지/5지), 단답형, 서술형, OX형 혼합 구성
4. **문항 작성**: Bloom's Taxonomy 수준별 문항 배분, 정답 및 해설 포함
5. **검토 및 오답 분석표**: 문항별 핵심 개념 연결, 오답 시 보충 학습 방향 제시

## 2026 한국 평가 트렌드 컨텍스트

### AI 기반 평가 자동화

에듀테크 코리아 페어 및 AI코스웨어 동향 반영:

- **자동 채점**: AI가 단답형·서술형 문항 자동 채점 및 피드백 제공
- **문항 자동 생성**: AI가 교육 목표에 맞는 문항 생성 (단순 기억 → 고차원 사고 문항 포함)
- **개인별 오답 클리닉**: 학습자별 오답 패턴 분석 → 맞춤형 보충 문제 추천
- **역량 맵핑**: 문항별 Bloom's Taxonomy 수준 레이블링으로 학습 깊이 시각화

### 한국 주요 시험별 특성

| 시험 | 특징 | AI 활용 포인트 |
|------|------|---------------|
| 수능 | 국·수·영·탐구 선택형+서술형 혼합 | 기출 패턴 분석, 약점 진단 |
| 공무원 시험 | 5지선다, 빠른 정확도 | 반복 암기 자동화, 모의고사 |
| TOEIC/TOEFL | 리딩·리스닝·스피킹·라이팅 | AI 발음 교정, 에세이 피드백 |
| 자격증 | 종목별 상이, 실기 포함 | 이론 문제 자동 생성, 실기 준비 체크리스트 |
| 내신 | 학교별 출제 기준 다양 | 교과서 기반 예상 문제, 수행평가 준비 |

## 문제 해결

| 상황 | 해결 방법 |
|------|-----------|
| 교육 자료 접근 불가 | 학습 목표와 핵심 개념 목록만 알려주시면 문항을 생성할 수 있습니다 |
| 학습 목표 불명확 | 시험 유형과 대상 수준을 먼저 확인하고 목표를 함께 설정합니다 |
| 특정 출판사 교과서 기반 출제 | 교과서 단원 제목과 핵심 내용을 입력해 주시면 그에 맞게 출제합니다 |
| 난이도 조절이 어려움 | 상/중/하 비율을 지정해 주시면 그에 맞춰 배분합니다 |
