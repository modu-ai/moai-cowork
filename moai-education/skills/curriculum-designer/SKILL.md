# 커리큘럼 디자이너 (Curriculum Designer)

> moai-education | 교육 설계 전문 스킬

## 전략 가이드

| ID | 한국명 | 설명 |
|----|--------|------|
| course-builder | 강의 빌더 | 커리큘럼 설계, 학습 목표, 온라인 강의 제작 |
| competency-modeler | 역량 모델러 | 직무 역량 분석, 스킬 갭 파악, 성장 경로 |
| language-tutor | 어학 튜터 | 외국어 학습 전략, 회화 연습, 문법 |

→ 참조 파일: `references/{id}.md`

## 트리거 키워드

강의, 커리큘럼, 온라인 교육, 학습 목표, 역량 모델, 스킬 갭, 외국어, 어학, 언어 학습

## 실행 규칙

1. 사용자 요청 수신 → 해당 전략 가이드 판별
2. `references/{id}.md` 로드 → 전략 가이드에 따라 실행
3. `--deepthink` 또는 복잡 학술 분석 → `mcp__sequential-thinking__sequentialthinking` 호출
4. 결과물 생성 후 사용자 검토 요청

## 사용 예시

- "Python 입문 온라인 강의 8주 커리큘럼을 만들어줘"
- "영업직 신입사원 역량 모델을 설계해줘"
- "TOEIC 900점을 위한 3개월 학습 계획을 세워줘"
- "사내 AI 활용 교육 프로그램을 기획해줘"
- "우리 팀 스킬 갭 분석하고 교육 로드맵 짜줘"

## 독립 실행 워크플로우

전략 가이드 파일 없이도 다음 단계로 실행 가능합니다:

1. **학습 목적 파악**: 대상 학습자(연령/직무/수준), 교육 목표, 기간 및 형식 확인
2. **학습 목표 정의**: Bloom's Taxonomy 6단계(기억→이해→적용→분석→평가→창조)에 따라 구체화
3. **커리큘럼 구조 설계**: 주차별/단원별 목차, 각 차시 학습 내용 및 활동 방식 설계
4. **평가 방법 결정**: 퀴즈, 과제, 프로젝트, 발표 등 학습 목표에 맞는 평가 방식 선정
5. **결과물 정리**: 학습 목표 명세서, 주차별 커리큘럼 표, 강사 가이드 초안 제공

## 2026 한국 교육 트렌드 컨텍스트

에듀모닝 2026-2027 교육산업 전망 보고서 및 에듀테크 코리아 페어 동향 반영:

### 핵심 키워드

- **초개인화 학습(Hyper-Personalized Learning)**: PwC 삼일 Paradigm Shift Vol.6 — 에듀테크의 핵심 가치는 학습자 개별 분석을 통한 1인 맞춤형 학습
- **AI 코스웨어**: 디지털 교과서 + AI 결합. 자동채점, 오답 클리닉, 학습 분석 대시보드 통합
- **실감화(VR/AR)**: 가상·증강현실 기반 몰입형 학습 콘텐츠 확산
- **연결화**: 학습자-교사-학부모-플랫폼 간 데이터 연결 생태계
- **융합화**: 교과 경계 초월 프로젝트 기반 학습(PBL) 확산

### 한국 커리큘럼 설계 시 고려사항

| 영역 | 2026 트렌드 |
|------|------------|
| 공교육 | AI 디지털 교과서 전면 도입 대비, 교사 역할 재정의 |
| 사교육 | AI 튜터 경쟁력 강화, 인간 강사의 고차원 사고 지도 차별화 |
| 기업 교육 | AX(AI Transformation) 역량 개발, 직무별 AI 활용 교육 |
| 어학 교육 | AI 발음 교정, 실시간 번역, 원어민 AI 대화 파트너 |

### Bloom's Taxonomy 적용 커리큘럼 설계

커리큘럼 목표를 Bloom's 6단계에 맞춰 구조화:
1. 기억 → 이해: 기본 개념 학습 (AI 자동화 영역)
2. 적용 → 분석: 실습·프로젝트 (교사 지도 영역)
3. 평가 → 창조: 비판적 사고·창작 (고차원 목표)

## 문제 해결

| 상황 | 해결 방법 |
|------|-----------|
| 교육 자료 접근 불가 | 학습자 수준, 목표 역량, 교육 기간을 알려주시면 구조만으로 설계 가능합니다 |
| 학습 목표 불명확 | "이 교육을 마친 학습자가 무엇을 할 수 있어야 하나요?"를 함께 논의해 드립니다 |
| 기존 커리큘럼 개선 요청 | 현행 목차나 강의계획서를 공유해 주시면 개선안을 제안해 드립니다 |
| 특수 교육 대상 (장애, 고령층 등) | 접근성 가이드라인과 유니버설 디자인 원칙을 적용해 드립니다 |
