# moai-finance

재무/세무 플러그인 — 3.3% 원천징수, 부가세, 홈택스, K-IFRS, 연말정산.

국내 세법 체계에 맞춘 세무 상담부터 재무제표 관리, 무역/SCM, 예산 분석까지 재무 실무 전반을 지원합니다.

## 스킬

| 스킬 | 설명 | 상태 |
|------|------|:----:|
| [tax-helper](./skills/tax-helper/) | 세무 도우미 — 3.3% 원천징수, 종합소득세, 부가가치세, 홈택스 안내 | ✅ 완성 |
| [financial-statements](./skills/financial-statements/) | 재무제표 관리 — 청구서/인보이스 관리, 정부 보조금 신청, 비영리 재무 관리 | ✅ 완성 |
| [close-management](./skills/close-management/) | 결산 및 무역 관리 — 수출입 실무, 통관, HS 코드, SCM, 나라장터 입찰 제안서 | ✅ 완성 |
| [variance-analysis](./skills/variance-analysis/) | 분산 분석 — 예산 대비 실적 차이, 원가 분석, 재무 KPI 추적, 수익성 개선 | 📝 초안 |

## 사용 예시

```
프리랜서 디자이너에게 200만원 지급할 때 3.3% 원천징수 어떻게 처리해?
```

```
2분기 영업이익이 목표 대비 15% 미달했어. 원인 분석 보고서 써줘.
```
