# 카테고리 4: 제품 & 데이터

## 카테고리 개요

**대분류**: 비즈니스 & 전략
**카테고리명**: 제품 & 데이터
**하네스 수**: 12개
**주요 특징**: 제품 개발, 데이터 기반 의사결정, 사용자 중심 설계

### 소개

제품 & 데이터 카테고리는 제품 전략부터 개발, 출시, 분석까지 제품 라이프사이클 전체를 다룹니다. 사용자 조사, UX 감사, 제품 분석, 데이터 시각화를 통해 데이터 기반의 제품 의사결정을 지원합니다.

### 주요 용도

- 사용자 중심 제품 개발
- 제품 로드맵 우선순위화
- 데이터 기반 의사결정
- 제품 성과 분석 및 최적화
- 대시보드 및 메트릭 정의

## 선택 기준

### 이 카테고리가 적합한 경우

- 제품 개발 또는 개선 진행 중
- 사용자 행동을 기반으로 한 의사결정 필요
- 제품 성과를 데이터로 측정하고 최적화
- 제품 전략 수립 또는 재정비
- 대시보드 또는 분석 환경 구축

### 관련 역할

- 제품 매니저
- UX/UI 디자이너
- 데이터 분석가
- 제품 소유자
- 제품 운영 담당

### 권장 시간 범위

- **즉시**: 제품 분석, UX 감사, A/B 테스트
- **단기 (1개월)**: 사용자 조사, 피드백 루프, 메트릭 정의
- **중기 (분기)**: 기능 우선순위, 제품 출시
- **장기**: 제품 전략, 로드맵 수립
