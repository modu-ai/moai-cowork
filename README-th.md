## 🎯 MoAI คืออะไร?

**MoAI-Cowork** เป็นระบบการเรียนรู้อัตโนมัติที่เปลี่ยน AI ให้เป็นผู้เชี่ยวชาญเฉพาะด้านสำหรับแต่ละขอบเขตขององค์กรของคุณ

- **100 เครื่องมือ**: 10 หมวดหมู่ × 10 สาขา (บัญชี กฎหมาย HR เทคโนโลยี การตลาด ฯลฯ)
- **วิวัฒนาคนเอง**: วงจร Self-Refine ตามข้อเสนอแนะของผู้ใช้และการวิเคราะห์ผลลัพธ์
- **โปรไฟล์ทั่วโลก**: การตอบสนองที่ปรับแต่งเฉพาะตัวผ่านโปรไฟล์บุคคล ทีม และองค์กร
- **ใช้ plugin**: การรวมกลมกลืนเด่นชัดกับ Claude Code และระบบนิเวศ Cowork

### ลักษณะสำคัญ

| ฟังก์ชั่น | คำอธิบาย |
|---------|---------|
| **การเรียนรู้อัตโนมัติ** | การบันทึกและวิเคราะห์การทำงานในแต่ละการโต้ตอบ |
| **ความเชี่ยวชาญด้านสาขา** | ความรู้เชิงปฏิบัติที่ล้ำสมัยในแต่ละสาขา |
| **การปรับตัวทางวัฒนธรรม** | สนับสนุนแนวปฏิบัติทางการค้าโลกและภาษา |
| **อัปเดตแบบเรียลไทม์** | รวมกฎหมายภาษีล่าสุด ข้อบังคับ และข้อมูลตลาด |
| **รองรับผู้ใช้หลายคน** | การจัดการโปรไฟล์ระดับทีมและการทำงานร่วมกัน |

## 📦 การติดตั้ง

### ข้อกำหนด
- Claude Code (เวอร์ชันล่าสุด)
- การรองรับ plugin Cowork

### ขั้นตอนที่ 1: เพิ่ม plugin ให้กับ Cowork
```bash
# ติดตั้งผ่าน Cowork CLI (ใกล้เข้ามา)
cowork install moai-cowork
```

### ขั้นตอนที่ 2: เริ่มต้นโปรไฟล์
```bash
moai init
```

### ขั้นตอนที่ 3: ตั้งค่าโปรไฟล์ส่วนบุคคล
```bash
moai profile --set-personal
```

## 🚀 เริ่มต้นอย่างรวดเร็ว

### เรียกใช้คำค้นหาแรกของคุณ

```
@moai ปฏิทินการประกาศภาษีมูลค่าเพิ่มของไทยในปี 2026 คืออะไร?
```

**ตอบสนอง**: MoAI จะโดยอัตโนมัติ:
1. โหลดข้อมูลการระบุตำแหน่งของไทย
2. ปรึกษาข้อมูลกฎหมายภาษี 2026
3. ให้ปฏิทินภาษีมูลค่าเพิ่มแบบปรับแต่งเฉพาะตัว

### เปิดใช้งานการเรียนรู้อัตโนมัติ

```
moai learn --feedback "คำตอบนั้นถูกต้องมาก"
```

MoAI บันทึกข้อเสนอแนะนี้เพื่อปรับปรุงคำค้นหาที่คล้ายกันในอนาคต

## 📚 ทำเนียบ 100 เครื่องมือ

### 10 หมวดหมู่

#### 1️⃣ ภาษีและบัญชี (Tax & Accounting)
- **TH_TAX_001**: ภาษีมูลค่าเพิ่มของไทย
- **US_TAX_001**: ภาษีรายได้ของสหรัฐฯ
- **JP_TAX_001**: ภาษีบริโภคของญี่ปุ่น
- **UK_TAX_001**: ภาษีมูลค่าเพิ่มของสหราชอาณาจักร
- **VN_TAX_001**: ภาษีมูลค่าเพิ่มของเวียดนาม
- **TH_TAX_001**: ภาษีมูลค่าเพิ่มของไทย
- *(6 เพิ่มเติม)*

#### 2️⃣ กฎหมายแรงงานและทรัพยากรบุคคล (Labor & HR)
- **TH_HR_001**: กฎหมายแรงงานของไทย
- **US_HR_001**: FLSA และค่าจ้างขั้นต่ำ
- **JP_HR_001**: กฎหมายแรงงานของญี่ปุ่น
- **UK_HR_001**: กฎหมายการจ้างงานของสหราชอาณาจักร
- *(6 เพิ่มเติม)*

#### 3️⃣ ข้อมูลและการปฏิบัติตามข้อกำหนด (Data & Compliance)
- **TH_DATA_001**: การคุ้มครองข้อมูลส่วนบุคคล
- **US_DATA_001**: CCPA/HIPAA
- **JP_DATA_001**: กฎหมายคุ้มครองข้อมูลส่วนบุคคล
- **UK_DATA_001**: UK GDPR
- *(6 เพิ่มเติม)*

#### 4️⃣ ปฏิบัติการทางธุรกิจ (Business Operations)
- **TH_BIZ_001**: แนวปฏิบัติทางธุรกิจของไทย
- **US_BIZ_001**: วัฒนธรรมการประชุมของอเมริกา
- **JP_BIZ_001**: วัฒนธรรมความร่วมมือของญี่ปุ่น
- *(7 เพิ่มเติม)*

#### 5️⃣ เทคโนโลยีและไอที (Technology & IT)
- **TECH_001**: แนวปฏิบัติที่ดีที่สุดในการพัฒนาซอฟต์แวร์
- **TECH_002**: สถาปัตยกรรมคลาউด์
- *(8 เพิ่มเติม)*

#### 6️⃣ การตลาดและการขาย (Marketing & Sales)
- **MKT_001**: กลยุทธ์การตลาดดิจิทัล
- **MKT_002**: เทคนิคการขาย B2B
- *(8 เพิ่มเติม)*

#### 7️⃣ การเงินและการลงทุน (Finance & Investment)
- **FIN_001**: การวิเคราะห์งบการเงิน
- **FIN_002**: การจัดการพอร์ตโฟลิโอการลงทุน
- *(8 เพิ่มเติม)*

#### 8️⃣ กฎหมายและสัญญา (Legal & Contracts)
- **LEG_001**: การตรวจสอบสัญญา
- **LEG_002**: การเขียน NDA
- *(8 เพิ่มเติม)*

#### 9️⃣ กลยุทธ์และการวางแผน (Strategy & Planning)
- **STR_001**: การสร้างกลยุทธ์ทางธุรกิจ
- **STR_002**: การตั้งค่า OKR
- *(8 เพิ่มเติม)*

#### 🔟 ลูกค้าและบริการ (Customer & Service)
- **CUS_001**: การวิเคราะห์ความพึงพอใจของลูกค้า
- **CUS_002**: แผนการปรับปรุงบริการ
- *(8 เพิ่มเติม)*

## 🔄 วงจรการเรียนรู้ตนเอง (Self-Refine)

### ลัดการไหลของการเรียนรู้

```
1. เรียกใช้คำค้นหา
   ↓
2. สร้างการตอบสนอง (โดยใช้เครื่องมือ)
   ↓
3. เก็บรวบรวมข้อเสนอแนะของผู้ใช้
   ↓
4. วิเคราะห์ผลลัพธ์ (ความถูกต้อง ความเกี่ยวข้อง ประโยชน์)
   ↓
5. ปรับปรุงเครื่องมือ (ปรับค่าน้ำหนัก)
   ↓
6. ใช้กับคำค้นหาถัดไป
```

### ประเภทของข้อเสนอแนะ

| ประเภท | คำอธิบาย | ผลกระทบ |
|--------|---------|--------|
| **ในเชิงบวก** | "ถูกต้องมาก" | น้ำหนักเครื่องมือ +10% |
| **บางส่วน** | "ถูกต้องบางส่วน" | น้ำหนัก ±5% |
| **เชิงลบ** | "ไม่ถูกต้อง" | น้ำหนักเครื่องมือ -15% |
| **กำหนดเอง** | "ต้องการเนื้อหาเพิ่มเติมเกี่ยวกับ X" | การเสริมสร้างสาขาเฉพาะ |

## 📖 ตัวอย่างการใช้งาน

### ตัวอย่างที่ 1: ปรึกษาภาษี
```
ถาม: "ขั้นตอนการลงทะเบียนเงินสมทบสำหรับพนักงานใหม่ในไทยคืออะไร?"
→ เปิดใช้งานเครื่องมือ TH_HR_001
→ ให้วันครบกำหนด เอกสารที่จำเป็น และขั้นตอน
```

### ตัวอย่างที่ 2: ธุรกิจระหว่างประเทศ
```
ถาม: "ฉันควรระวังอะไรในการประชุมทางธุรกิจในอเมริกา?"
→ เปิดใช้งานเครื่องมือ US_BIZ_001
→ คำแนะนำการปรับตัวทางวัฒนธรรม การจัดการเวลา สไตล์การสื่อสาร
```

### ตัวอย่างที่ 3: การปฏิบัติตามข้อบังคับ
```
ถาม: "ฉันต้องทำตามขั้นตอนใดบ้างในการประมวลผลข้อมูลลูกค้าในสหภาพยุโรป?"
→ เปิดใช้งานเครื่องมือ TH_DATA_001 และ GDPR ที่เกี่ยวข้อง
→ การปฏิบัติตาม GDPR การจัดการความยินยอม ขั้นตอนการโอนข้อมูล
```

## 🛠 วิธีการมีส่วนร่วม (Contributing)

### 1. เสนอเครื่องมือใหม่
```bash
# เสนอสาขาใหม่
moai contribute --domain "กฎหมายการค้าของไทย" --category "legal"
```

### 2. ปรับปรุงเครื่องมือที่มีอยู่
```bash
# การปรับปรุงตามข้อเสนอแนะ
moai improve DOMAIN_ID --feedback "ต้องการเนื้อหาเพิ่มเติม"
```

### 3. เพิ่มการแปลเป็นภาษาท้องถิ่นใหม่
```bash
# เพิ่มการแปลเป็นภาษาท้องถิ่นประเทศใหม่
moai add-locale --country "กัมพูชา" --code "km_KH"
```

### 4. ปรับปรุงเอกสาร
- แก้ไขไฟล์ที่แปลเป็นภาษาท้องถิ่นใน `/skills/moai/references/locale/`
- ส่ง Pull Request บน GitHub
- อัปเดตคำแนะนำการปรับตัวทางวัฒนธรรม
