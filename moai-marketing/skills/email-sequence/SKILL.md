# 이메일 시퀀스 (Email Sequence)

> moai-marketing v1.0.0

## 개요

한국 정보통신망법(제50조) 수신동의 요건을 완전 준수하면서 비즈니스 목표를 달성하는 이메일 시퀀스를 설계한다.
드립 캠페인, 고객 온보딩, 재활성화, 뉴스레터 등 다양한 시퀀스 유형을 커버하며, 필수 법적 요소를 자동으로 포함한다.

## 트리거 키워드

이메일 시퀀스, 드립 캠페인, 이메일 자동화, 뉴스레터, 온보딩 이메일, 재활성화 이메일, 이메일 마케팅, 이메일 플로우, 리드 너처링

## 워크플로우

### 1단계: 시퀀스 목적 및 타겟 확인
- 시퀀스 유형 선택:
  - 웰컴/온보딩 (신규 가입자)
  - 드립 육성 (리드 → 전환)
  - 재활성화 (이탈 고객)
  - 프로모션 (시즌 캠페인)
  - 뉴스레터 (정기 콘텐츠)
- 타겟 세그먼트 정의 (업종, 직군, 행동 기반)
- 전환 목표 설정 (구매, 상담 신청, 콘텐츠 소비 등)

### 2단계: 법적 컴플라이언스 체크 (필수)

**정보통신망법 제50조 준수 필수 항목:**
- [ ] 수신자가 광고성 정보 수신에 **명시적으로 사전 동의** 했는지 확인
  - 서비스 가입만으로는 동의 불충분 — 광고 수신 별도 동의 필요
  - 개인정보 수집·이용 동의와 **별도** 광고 수신 동의 필요
- [ ] 이메일 제목 시작 부분에 **`(광고)`** 표시 의무
  - 제목 중간·끝·본문에 표시하면 위반 (3천만원 이하 과태료)
- [ ] 본문에 **전송자 명칭·주소·전화번호** 명시
- [ ] **무료 수신거부 방법** 안내 필수
  - "무료수신거부" 링크 또는 회신 이메일 제공
- [ ] 수신거부 처리 시 결과 통지 의무
- [ ] **정기적 수신동의 여부 재확인** 의무 (2024년 개정 사항)

### 3단계: 시퀀스 구조 설계
```
시퀀스 예시: SaaS 온보딩 7단계

D+0: 웰컴 이메일 (즉시 발송)
  → 핵심 가치 제안 + 첫 액션 유도

D+1: 시작 가이드
  → 핵심 기능 1가지 집중 소개

D+3: 성공 사례
  → 유사 고객 케이스 스터디

D+7: 심화 기능 소개
  → 업그레이드 유도 (소프트 CTA)

D+14: 체크인
  → 사용 현황 확인 + 장벽 제거

D+21: 사회적 증명
  → 리뷰/추천사 + 한정 프로모션

D+30: 전환 유도
  → 강력한 CTA + 보너스 제안
```

### 4단계: 이메일 본문 작성

**이메일 구조 (AIDA 원칙):**
- **제목**: 30자 이내, 열람률 최적화, `(광고)` 필수 포함
- **프리헤더**: 제목 보완 정보 (40자 이내)
- **헤더**: 브랜드 로고 + 핵심 메시지
- **본문**: 단일 목적 집중, 500자 이내 권장
- **CTA**: 버튼 1개 원칙, 명확한 행동 지시
- **푸터**: 전송자 정보, 수신거부 링크, 개인정보처리방침

**이메일 마케팅 카피 팁:**
- 존댓말 사용 (합쇼체/해요체 선택 일관성 유지)
- 혜택을 수치로 표현 (예: "30% 절감", "2시간 단축")
- 긴급성·희소성 문구는 사실에 근거할 것

### 5단계: 성과 지표 설정 및 A/B 테스트 계획
- 오픈율 목표: 업종별 기준 (B2B 25~35%, B2C 15~25%)
- 클릭률 목표: 2.5~5%
- 전환율 목표: 시퀀스 유형별 설정
- A/B 테스트 항목: 제목, 발송 시간, CTA 버튼 문구

## 사용 예시

**예시 1**: "SaaS 제품 신규 가입자 온보딩 이메일 7단계 시퀀스 만들어줘"
→ 법적 체크 → 시퀀스 구조 → 각 단계 이메일 초안 → 발송 스케줄

**예시 2**: "90일 이상 미활성 고객 재활성화 드립 캠페인 3단계 설계해줘"
→ 세그먼트 정의 → 재활성화 시퀀스 → 인센티브 제안 → 최종 수신거부 처리

## 산출물

- 시퀀스 플로우 다이어그램 (단계별 트리거·조건)
- 각 이메일 본문 초안 (제목 포함)
- 정보통신망법 컴플라이언스 체크리스트
- 성과 측정 KPI 대시보드 설계
- 수신거부 처리 프로세스 문서

## 참고 사항

### 정보통신망법 핵심 요약 (2026 기준)
- **근거법**: 정보통신망 이용촉진 및 정보보호 등에 관한 법률 제50조
- **최신 개정**: 2024년 6차 개정판 (KISA 불법스팸 방지 안내서 반영)
- **위반 시 제재**: 사전 동의 없는 광고 발송 시 **3천만원 이하 과태료**
- **단체 발송 시 개인정보 유출 주의**: 수신자 이메일 타인 공개 시 **5천만원 이하 과태료**
- **정기 재동의 확인**: 장기 미활성 구독자에게 정기적으로 수신 의사 재확인 필수

### 추천 이메일 마케팅 도구
- **스티비 (Stibee)**: 국내 정보통신망법 자동 준수 기능 내장, 한국어 지원 최적화
- **mailchimp**: 글로벌 표준, 대용량 발송
- **ActiveCampaign**: 자동화 복잡 시퀀스에 강점
- **채널톡**: 한국 스타트업 특화 인앱+이메일 통합

### 발송 최적 시간 (B2B 기준)
- 화·수·목요일 오전 9시~11시 (오픈율 최고)
- 점심 시간대 (12~13시) 재오픈율 높음
- 금요일 오후·월요일 아침 발송 비추천

## 문제 해결

| 상황 | 대응 |
|------|------|
| 고객 데이터나 세그먼트 정보가 없는 경우 | 업종·목표 기반 일반적 시퀀스 구성. 실제 데이터 확보 후 세분화 권고 |
| 법적 요구사항이 불명확한 경우 | 보수적 기준(별도 광고 수신 동의)으로 설계. 법무팀 검토 권고 |
| 발송 도구가 정해지지 않은 경우 | 규모·예산 기반 도구 추천 후 설계 방향 맞춤 조정 |

## 이 스킬을 사용하지 말아야 할 때

- SNS 콘텐츠나 소셜 미디어 캠페인 → moai-marketing/sns-content 또는 campaign-planner 사용
- 마케팅 성과 분석 보고서 → moai-marketing/performance-report 사용
- 법적 스팸 여부 판단이나 분쟁 대응 → 전문 법무팀에 의뢰하세요
- 단순 이메일 초안 작성 → 일반 대화로 해결
