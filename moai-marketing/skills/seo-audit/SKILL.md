# SEO 감사 (SEO Audit)

> moai-marketing v1.0.0

## 개요

네이버·구글 양대 검색엔진과 ChatGPT·Perplexity 등 AI 검색(GEO)까지 포괄하는 통합 SEO 감사 및 최적화 전략을 제공한다.
2026년 핵심 트렌드인 네이버 스마트블록·C-Rank 전문성, 구글 AI Overview, GEO(생성형 검색 최적화)를 반영한다.

## 트리거 키워드

SEO 감사, 검색최적화, 키워드 분석, 네이버 SEO, 구글 SEO, 온페이지 SEO, 백링크, 검색 순위, C-Rank, DIA, GEO, AI 검색 최적화, 상위노출

## 워크플로우

### 1단계: 감사 범위 확인
- 대상 URL / 도메인 수집
- 목표 플랫폼 선택: 네이버 / 구글 / AI 검색(GEO) / 전체
- 주요 타겟 키워드 3~5개 확인

### 2단계: 네이버 SEO 감사 (C-Rank + D.I.A.)
**C-Rank 평가 항목 (블로그/사이트 신뢰도)**
- 주제 집중도: 31개 카테고리 중 단일 주제 전문성 확인
- 콘텐츠 발행 일관성 및 발행 주기
- 사용자 반응 지표 (공감, 댓글, 스크랩, 체류시간)
- VIEW 통합 노출 여부 (블로그·카페 스마트블록 노출 확인)

**D.I.A. / D.I.A.+ 평가 항목 (문서 품질)**
- 제목-본문 키워드 적합도
- 경험 기반 정보 포함 여부 (후기, 실제 사용 경험)
- 글의 구조 (소제목, 이미지, 표 구성)
- 검색 의도(Query Intent) 부합도
- AI 생성 콘텐츠 패널티 리스크 평가

**2026년 네이버 최신 변화 체크**
- [ ] 스마트블록 노출 구조 확인
- [ ] AI 콘텐츠 탐지 패널티 리스크
- [ ] 네이버TV 영상 콘텐츠 연동 여부
- [ ] 로컬 SEO (지역 특화 키워드) 대응

### 3단계: 구글 SEO 감사 (Core Web Vitals + E-E-A-T)
**기술 SEO**
- Core Web Vitals: LCP < 2.5s, FID < 100ms, CLS < 0.1
- 모바일 친화성 (Mobile-First Index)
- 사이트맵·robots.txt 설정
- HTTPS 보안 인증

**콘텐츠 SEO (E-E-A-T)**
- Experience (경험) · Expertise (전문성) · Authoritativeness (권위) · Trustworthiness (신뢰성)
- 구조화 데이터 (Schema.org) 마크업

### 4단계: GEO 감사 (생성형 AI 검색 최적화) — 2026 신규
**AI 검색 엔진 크롤러 접근성**
- GPTBot, ClaudeBot, PerplexityBot robots.txt 차단 여부 확인
- llms.txt 파일 존재 여부 (AI 크롤러 안내)

**AI 인용 최적화**
- FAQ, HowTo, Article 스키마 마크업
- 명확한 답변형 문단 구조 (질문→답변 형식)
- 신뢰할 수 있는 데이터·출처 포함 여부

**2026년 GEO 지표**
- Perplexity·Google AIO 인용 여부 (최적화 후 2~4주 내 확인)
- ChatGPT·Gemini 브랜드 언급 (2~3개월 관찰)

## 참고 사항

### 네이버 SEO 핵심 원칙 (2026)
- C-Rank 상위 노출을 위해 **단일 주제 전문 블로그** 운영이 가장 중요
- 규칙적인 콘텐츠 발행 주기 유지 (주 2~3회 권장)
- AI 생성 글 남용 시 패널티 위험 — 경험 기반의 오리지널 콘텐츠 필수
- 해시태그보다 본문 키워드 배치가 더 중요

### GEO 전략 핵심 (2026)
- 가트너 예측: 2026년 말 오가닉 검색 트래픽 25% 감소 전망
- SEO + GEO 병행 전략 필수: "구글 순위"와 "AI 답변 인용"은 별개의 게임
- 네이버 생태계 + AI 응답 이중 최적화 전략이 한국 시장 핵심
