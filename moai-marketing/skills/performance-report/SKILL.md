# 퍼포먼스 리포트 (Performance Report)

> moai-marketing v1.0.0

## 개요

GA4, 네이버 광고, 메타(Meta) 광고, 카카오모먼트, 구글 광고 등 한국 시장 주요 채널의 성과 데이터를 통합 분석하여 경영진과 실무팀 모두가 이해할 수 있는 마케팅 성과 보고서를 생성한다.
데이터 해석과 인사이트 도출, 다음 액션 플랜까지 포함한 완성형 리포트를 제공한다.

## 트리거 키워드

성과 보고서, KPI 리포트, ROAS 분석, 광고 성과, GA4 분석, 메타 광고 리포트, 네이버 광고 분석, 퍼포먼스 리포트, 마케팅 ROI, 전환율 분석, CTR, CPC, 광고비 분석

## 워크플로우

### 1단계: 보고서 범위 설정
- 보고 기간 선택 (주간/월간/분기/연간)
- 포함할 채널 선택:
  - 유료 광고: 네이버 광고, 구글 광고(GDN+검색), 메타(FB/IG), 카카오모먼트, 쿠팡 광고
  - 유기적 채널: 유기 검색, SNS 오가닉, 이메일, 추천
  - 분석 도구: GA4, 네이버 애널리틱스, 픽셀 데이터
- 목표 KPI 확인 (매출, 리드, 앱 설치, 브랜드 인지 등)

### 2단계: 핵심 지표 수집 가이드
사용자가 제공할 데이터 형식 안내:

**필수 데이터 (GA4 기준):**
- 세션 수, 사용자 수, 신규 사용자 비율
- 전환수, 전환율, 전환 가치
- 채널별 획득 데이터 (Source/Medium/Campaign)
- 이탈률, 세션 지속 시간, 페이지뷰

**광고 플랫폼 데이터:**
| 지표 | 정의 | 목표 기준 |
|------|------|-----------|
| ROAS | 광고 수익 / 광고비 × 100 | 업종별 상이 (e-commerce 300~500% 일반적) |
| CPA | 광고비 / 전환수 | 고객 LTV의 1/3 이하 권장 |
| CTR | 클릭수 / 노출수 × 100 | 검색 2%+, 디스플레이 0.1%+ |
| CPC | 광고비 / 클릭수 | 업종별 상이 |
| CVR | 전환수 / 클릭수 × 100 | 랜딩페이지 최적화 여부 판단 |
| CPM | 광고비 / 노출수 × 1,000 | 브랜드 인지도 캠페인 기준 |

### 3단계: 채널별 분석

**네이버 광고 분석 포인트:**
- 파워링크 vs 쇼핑 광고 성과 비교
- 스마트플레이스 노출 현황
- 네이버 쇼핑 ROAS (상품별)
- C-Rank 영향 유기 트래픽 변화

**메타(인스타그램/페이스북) 광고 분석:**
- 2026년 알고리즘 변화 반영:
  - 릴스 vs 피드 광고 성과 비교 (릴스 평균 30% 높은 참여율)
  - DM·공유·저장 등 딥 인게이지먼트 지표 확인
- 오디언스 인사이트 (나이·성별·지역·관심사)
- 광고 소재 A/B 테스트 결과

**구글 광고:**
- 검색 캠페인 품질점수 (Quality Score)
- 전환 추적 (GA4 연동 확인)
- 스마트 캠페인 vs 수동 캠페인 성과

### 4단계: 보고서 생성 (표준 포맷)

```markdown
# [기간] 마케팅 성과 보고서

## 경영진 요약 (Executive Summary)
- 총 광고비: xxx원 / 총 매출: xxx원 / 전체 ROAS: xxx%
- 전기 대비: ▲/▼ xx%
- 핵심 인사이트 3줄 요약

## 채널별 성과표
| 채널 | 노출 | 클릭 | CTR | 전환 | ROAS |

## 주요 발견 및 인사이트

## 개선 기회 (Opportunities)

## 다음 기간 액션 플랜
- 예산 조정 권고
- 크리에이티브 개선 방향
- 테스트 제안
```

### 5단계: 인사이트 도출 및 권고안
- 성과 부진 채널 원인 분석 (예산? 타겟? 소재? 랜딩페이지?)
- 예산 재배분 시뮬레이션
- 다음 기간 테스트 가설 제안

## 사용 예시

**예시 1**: "지난달 네이버+메타 광고 성과 통합 분석해줘. 데이터 여기 있어: [데이터]"
→ 데이터 파싱 → 채널별 KPI → 비교 분석 → 인사이트 → 다음 달 권고안

**예시 2**: "분기 마케팅 성과 경영진 보고서 만들어줘. 총 광고비 3천만원, ROAS 350%"
→ Executive Summary → 채널별 성과표 → 주요 발견 → 다음 분기 전략

## 산출물

- 채널 통합 성과 대시보드 (마크다운/표 형식)
- 경영진용 요약 리포트 (1페이지)
- 실무팀용 상세 분석 리포트
- 다음 기간 예산 배분 권고안
- A/B 테스트 결과 분석

## 참고 사항

### 한국 주요 광고 플랫폼
- **네이버 광고**: 국내 검색 점유율 1위, 쇼핑 광고 강력
- **카카오모먼트**: 카카오톡 기반, 모바일 타겟팅 강점
- **메타(인스타/페북)**: 2026년 릴스 중심 개편, 쿠팡 연동 부상
- **쿠팡 로켓그로스**: 이커머스 특화, ROAS 추적 가능
- **구글**: 검색+유튜브 영상 광고 통합 관리

### 2026년 주목 지표
- **AI 검색 트래픽**: Perplexity, ChatGPT 등 AI 검색 유입 별도 추적 시작 권고
- **제로클릭 증가**: AI 답변에서 직접 해결 → 사이트 유입 감소 모니터링 필요
- **딥 인게이지먼트**: 단순 클릭보다 저장·공유·체류시간이 핵심 지표로 부상

## 문제 해결

| 상황 | 대응 |
|------|------|
| 데이터가 없거나 불완전한 경우 | 보유한 데이터로 분석 가능한 범위 제시. 누락 데이터 수집 방법 안내 |
| 채널이 너무 많아 복잡한 경우 | 매출 기여도 상위 2~3개 채널 집중 분석 후 나머지 요약 처리 |
| 업종 벤치마크가 없는 경우 | 범용 업종 기준값 제시. 사용자 직접 측정 KPI 설정 권고 |

## 이 스킬을 사용하지 말아야 할 때

- 마케팅 캠페인 기획이나 새 캠페인 전략 수립 → moai-marketing/campaign-planner 사용
- SEO 감사나 검색 최적화 → moai-marketing/seo-audit 사용
- 재무 P&L 분석이나 회계 처리 → moai-finance 사용
- 단순 수치 계산이나 일반 질문 → 일반 대화로 해결
