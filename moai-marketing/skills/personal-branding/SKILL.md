# 개인 브랜딩 (Personal Branding)

> moai-marketing v1.0.0

## 개요

개인의 전문성, 경험, 가치관을 체계적으로 브랜드화하여 커리어와 사업 성장을 지원한다.
2026년 커리어 브랜딩 중심 트렌드, AI 활용 브랜딩, 숏폼+롱폼 하이브리드 전략을 반영한다.
개인 스토리텔링보다 구체적 성과와 전문성 기반의 커리어 브랜딩을 우선한다.

## 지원 영역

| 영역 | 설명 | 적합 대상 |
|------|------|-----------|
| 커리어 전환 | 새로운 직무/업계로의 브랜드 재구축 | 이직 준비자, 직무 전환자 |
| 전문가 포지셔닝 | 업계 내 전문가로서의 인지도 확보 | 현직 전문가, 컨설턴트 |
| 프리랜서 브랜딩 | 개인 사업자로서의 신뢰 구축 | 프리랜서, 1인 기업 |
| 1인 기업 | 사업 성장을 위한 대표자 브랜딩 | 1인 창업자, 소규모 사업자 |
| 강연자/저자 | 강연, 출판, 교육 시장 진출 전략 | 강연자, 작가 지망생 |
| 부업/사이드 프로젝트 | 본업과 병행하는 개인 브랜드 구축 | 직장인 부업, 사이드 허슬러 |

## 독립 실행 워크플로우

### 사전 수집: 개인 브랜드 인터뷰 체크리스트

실행 전 사용자에게 다음 정보를 수집한다:

```
[ ] 현재 직업/직함과 경력 연차
[ ] 핵심 전문 분야 (1~3개)
[ ] 대표 성과 또는 프로젝트 (구체적 수치 포함)
[ ] 타겟 오디언스 (누구에게 전문가로 인정받고 싶은가)
[ ] 브랜딩 목표 (이직, 강연 섭외, 클라이언트 확보, 인지도 등)
[ ] 현재 운영 중인 온라인 채널
[ ] 벤치마킹하고 싶은 전문가 (1~3명)
[ ] 월간 콘텐츠 투입 가능 시간
```

### 1단계: 자기 분석

**목표**: 강점, 전문성, 차별화 요소, 개인 가치 제안(PVP)을 도출한다.

**실행 항목:**
1. 전문성 인벤토리 작성 (기술, 경험, 자격, 성과 정리)
2. 핵심 가치 도출 (가치 카드 정렬법 활용)
3. 차별화 요소 발견 (3C 분석: 나-경쟁자-고객)
4. 개인 가치 제안(PVP) 공식 작성:
   "[타겟]이 [문제]를 겪을 때, 나는 [방법]으로 [결과]를 제공하는 [직함]이다"
5. 브랜드 키워드 3개 선정 (형용사 + 전문 분야 + 가치)

**산출물**: `self_assessment.md`

상세 프레임워크: `references/personal-branding/self-assessment-framework.md` 참조

### 2단계: 포지셔닝 전략

**목표**: 시장 내 최적 포지션을 설정하고 타겟 오디언스를 정의한다.

**실행 항목:**
1. 타겟 오디언스 페르소나 작성 (직업, 연령, 관심사, 고민, 소비 채널)
2. 경쟁 전문가 분석 (3~5명의 포지셔닝, 콘텐츠 전략, 강약점)
3. 포지셔닝 맵 작성:
   - X축: 접근성 (대중적 - 전문적)
   - Y축: 스타일 (감성적 - 논리적)
   - 경쟁자 배치 후 빈 공간 파악
4. 포지셔닝 스테이트먼트 작성:
   "[카테고리]에서 [타겟]을 위해 [차별화 요소]를 제공하는 [이름]"
5. 전문가 등급 목표 설정 (로컬 전문가 / 업계 전문가 / 사고 리더)

**산출물**: `brand_strategy.md`

### 3단계: 브랜드 아이덴티티

**목표**: 일관된 브랜드 이미지를 구축할 시각적/언어적 요소를 정의한다.

**실행 항목:**
1. 이름/별명 전략 (실명 vs 전문가명 vs 닉네임)
2. 태그라인 작성 (10자 이내, 가치 제안 압축):
   - 공식: "[핵심 동사] + [타겟의 결과]"
   - 예시: "데이터로 말하는 마케터", "코드로 문제를 푸는 사람"
3. 비주얼 톤 정의:
   - 프로필 사진 가이드 (배경, 의상, 표정, 각도)
   - 대표 컬러 2~3색 선정 (채널 통일)
   - 폰트/디자인 톤 (전문적/캐주얼/크리에이티브)
4. 프로필 최적화:
   - 통합 자기소개 (50자/150자/300자 버전)
   - 플랫폼별 프로필 최적화 체크리스트
5. 어조/톤 가이드:
   - 공식적-비공식적 스펙트럼에서 위치 설정
   - 사용할 어투 (존댓말/반말/혼합)
   - 자주 쓸 표현/절대 쓰지 않을 표현

**산출물**: `brand_identity.md`

### 4단계: 콘텐츠 전략

**목표**: 숏폼+롱폼 결합 콘텐츠 전략과 월간 캘린더를 수립한다.

**실행 항목:**
1. 핵심 주제(Pillar) 3개 선정
2. 하위 주제(Cluster) 매핑 (각 핵심 주제당 5~7개)
3. 숏폼+롱폼 결합 전략 수립:
   - 숏폼 (릴스/쇼츠): 주 2~3회, 인지도 확대
   - 롱폼 (블로그/유튜브): 주 1회, 전문성 입증
   - 연결 전략: 숏폼에서 롱폼으로 유도하는 CTA
4. 월간 콘텐츠 캘린더 작성 (주별 테마 + 플랫폼별 스케줄)
5. AI 활용 vs 직접 작성 영역 구분

**산출물**: `content_strategy.md`

상세 전략: `references/personal-branding/content-strategy.md` 참조

### 5단계: 채널 전략

**목표**: 플랫폼별 최적화 전략과 크로스 포스팅 체계를 수립한다.

**실행 항목:**
1. 주력 채널 선정 (최대 3개 — 집중력 유지)
2. 플랫폼별 프로필 최적화 실행
3. 크로스 포스팅 전략 수립 (1개 콘텐츠 -> 다수 플랫폼 변환)
4. 성과 측정 KPI 설정:
   - 도달: 노출수, 프로필 방문수
   - 참여: 좋아요율, 댓글수, 저장수, 공유수
   - 전환: DM 문의, 미팅 요청, 강연 섭외, 수익
5. 월간 성과 리뷰 체크리스트

**산출물**: `channel_strategy.md`

상세 가이드: `references/personal-branding/channel-guide.md` 참조

## 핵심 규칙

### 진정성 유지 원칙
- 과장이나 허위 경력을 포함하지 않는다
- 실제 경험과 성과에 기반한 브랜딩만 권장한다
- "보이는 것"보다 "증명할 수 있는 것"을 우선한다
- 겸양과 전문성 어필 사이의 균형을 유지한다

### AI 과의존 방지 원칙
- AI로 자동화할 것: 초안 작성, 캘린더 설계, 해시태그 추천, 이미지 생성
- 반드시 인간이 할 것: 개인 경험담, 고유 인사이트, 감정 표현, 관계 구축
- AI 생성 콘텐츠 비율: 전체 콘텐츠의 40% 이하로 유지한다
- AI 티 안 나는 콘텐츠 작성 5원칙:
  1. 구체적 에피소드 삽입
  2. 불완전한 문장/구어체 활용
  3. 개인 의견 명시적 표현
  4. 날짜/장소/인물 특정
  5. 독자 질문 유도

### 2026 커리어 브랜딩 원칙
- 개인 서사(나는 누구인가)보다 전문 성과(무엇을 해냈는가)를 중심으로 구성한다
- 라이프스타일 세그먼트 기반으로 타겟을 설정한다 (인구통계 기반 지양)
- 숏폼으로 주의를 끌고, 롱폼으로 깊이를 전달하는 하이브리드 전략을 기본으로 한다

## 실행 규칙

- 모든 산출물은 마크다운 형식으로 작성한다
- 각 단계 완료 후 사용자 확인을 받은 뒤 다음 단계로 진행한다
- 사전 수집 체크리스트의 필수 항목(직업, 전문 분야, 목표)이 누락된 경우 진행을 중단하고 수집한다
- 벤치마킹 전문가 분석 시 실존하지 않는 인물을 생성하지 않는다
- 통계나 시장 데이터 인용 시 출처와 기준 연도를 명시한다

## 이 스킬을 사용하지 말아야 할 때

- **기업/제품 브랜딩**: 회사나 상품의 브랜드 아이덴티티는 `brand-identity` 스킬이 더 적합하다.
- **SNS 콘텐츠 단독 작성**: 인스타 릴스, 링크드인 포스트 등 개별 콘텐츠 제작은 `social-media` 또는 `sns-content` 스킬을 사용한다.
- **블로그 포스팅 작성**: 개별 블로그 글 작성은 `blog` 스킬이 더 적합하다.
- **광고 캠페인 기획**: 유료 광고 전략은 `advertising-campaign` 스킬을 사용한다.
- **포트폴리오 사이트 제작**: 실제 웹사이트 구현은 `agency` 스킬을 사용한다.

## 문제 해결

- **전문 분야가 불명확한 경우**: 자기 분석 단계에서 전문성 인벤토리를 먼저 작성하고, 가장 많은 경험과 성과가 집중된 영역을 핵심 분야로 선정한다.
- **타겟 오디언스를 모르는 경우**: 현재까지 도움을 준 사람들의 공통점을 분석하여 타겟을 역추적한다.
- **콘텐츠 아이디어 고갈**: 핵심 주제 3개 x 하위 주제 5~7개 = 최소 15~21개 주제 풀을 먼저 확보한다. 댓글, DM, 자주 받는 질문에서 추가 아이디어를 발굴한다.
- **여러 채널을 동시에 운영하기 어려운 경우**: 주력 채널 1개에 집중하고, 크로스 포스팅으로 나머지 채널을 보조적으로 운영한다.
- **직장인이라 실명 노출이 부담되는 경우**: 전문가명 또는 닉네임 전략을 활용하고, 회사 관련 내용은 배제한 순수 전문 지식 중심으로 운영한다.
