# moai-marketing

마케팅 플러그인 — 네이버/카카오 SEO, SNS 콘텐츠, 캠페인 기획, 이메일 시퀀스.

국내 플랫폼(네이버 블로그, 카카오 채널, 인스타그램) 특성에 맞춘 콘텐츠 전략과 퍼포먼스 마케팅을 지원합니다.

## 스킬

| 스킬 | 설명 | 상태 |
|------|------|:----:|
| [sns-content](./skills/sns-content/) | SNS 콘텐츠 — 네이버 블로그, 인스타그램, 카카오 채널 최적화. 브랜드 아이덴티티, 퍼스널 브랜딩 | ✅ 완성 |
| [campaign-planner](./skills/campaign-planner/) | 캠페인 기획 — A/B 테스트, 그로스 해킹, 인플루언서 전략, CRM, 이커머스 상세페이지 | ✅ 완성 |
| [seo-audit](./skills/seo-audit/) | SEO 감사 — 키워드 분석, 네이버/구글 검색 최적화, 콘텐츠 SEO 전략 | 📝 초안 |
| [email-sequence](./skills/email-sequence/) | 이메일 시퀀스 — 드립 캠페인, 고객 온보딩 이메일, 재활성화 시퀀스 | 📝 초안 |
| [performance-report](./skills/performance-report/) | 퍼포먼스 리포트 — ROAS 분석, GA4/메타/네이버 광고 성과 KPI 대시보드 | 📝 초안 |

## 사용 예시

```
인스타그램용 브랜드 아이덴티티 카피 5개 만들어줘. 20-30대 여성 타깃, 친근한 톤.
```

```
네이버 블로그 SEO 최적화 전략 세워줘. 키워드는 "재테크 방법"이야.
```
