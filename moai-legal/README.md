# moai-legal

법률 플러그인 — 계약서 검토(민법/상법), 개인정보보호법 컴플라이언스, NDA, 지적재산.

> **주의**: 이 플러그인은 AI 보조 법률 분석 도구입니다. 중요한 법적 결정은 반드시 법률 전문가와 상담하십시오.

## 스킬

| 스킬 | 설명 | 상태 |

| [compliance-check](./skills/compliance-check/) | 컴플라이언스 점검 — 규제 준수 갭 분석, ESG 보고, 인허가 서류 지원 | ✅ 완성 |
| [legal-risk](./skills/legal-risk/) | 법적 리스크 분석 — 판례/법령 리서치, 특허·상표·저작권 IP 포트폴리오 관리 | ✅ 완성 |
