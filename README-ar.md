## 🎯 ما هو MoAI؟

**MoAI-Cowork** هو نظام تعلم آلي يحول الذكاء الاصطناعي إلى خبراء متخصصين لكل مجال في منظمتك.

- **100 أداة**: 10 فئات × 10 مجالات (المحاسبة والقانون والموارد البشرية والتكنولوجيا والتسويق وغيرها)
- **التطور الذاتي**: دورة Self-Refine بناءً على تعليقات المستخدم وتحليل النتائج
- **الملف الشامل العالمي**: استجابات مخصصة من خلال ملفات فردية وفريق وتنظيمية
- **قائم على المكونات الإضافية**: تكامل سلس مع Claude Code وبيئة Cowork

### الميزات الأساسية

| الوظيفة | الوصف |
|--------|--------|
| **التعلم الآلي** | تسجيل وتحليل الأداء في كل تفاعل |
| **التخصص في المجال** | معرفة عملية حديثة في كل مجال |
| **التكيف الثقافي** | دعم الممارسات التجارية العالمية واللغات |
| **التحديثات في الوقت الفعلي** | يتضمن أحدث قوانين الضرائب والتنظيمات ومعلومات السوق |
| **دعم متعدد المستخدمين** | إدارة الملفات الشخصية على مستوى الفريق والتعاون |

## 📦 التثبيت

### المتطلبات
- Claude Code (أحدث إصدار)
- دعم مكون إضافي Cowork

### الخطوة 1: إضافة المكون الإضافي إلى Cowork
```bash
# التثبيت عبر Cowork CLI (قريباً)
cowork install moai-cowork
```

### الخطوة 2: تهيئة الملف الشخصي
```bash
moai init
```

### الخطوة 3: تكوين ملفك الشخصي
```bash
moai profile --set-personal
```

## 🔄 دورة التعلم الذاتي (Self-Refine)

### تدفق التعلم

```
1. تشغيل الاستعلام
   ↓
2. إنشاء الرد (باستخدام الأدوات)
   ↓
3. جمع تعليقات المستخدم
   ↓
4. تحليل النتائج (الدقة والملاءمة والفائدة)
   ↓
5. تحسين الأداة (ضبط الأوزان)
   ↓
6. التطبيق على الاستعلام التالي
```

### أنواع التعليقات

| النوع | الوصف | التأثير |
|-------|-------|---------|
| **إيجابي** | "دقيق جداً" | وزن الأداة +10% |
| **جزئي** | "جزئياً صحيح" | الوزن ±5% |
| **سلبي** | "غير صحيح" | وزن الأداة -15% |
| **مخصص** | "يحتاج إلى مزيد من المحتوى حول X" | تعزيز منطقة معينة |

## 📖 أمثلة الاستخدام

### المثال 1: استشارة ضريبية
```
س: "ما هي عملية تسجيل الاشتراكات للموظفين الجدد في المملكة العربية السعودية؟"
→ تفعيل الأداة SA_HR_001
→ توفير المواعيد والمستندات المطلوبة والإجراء
```

### المثال 2: الأعمال التجارية الدولية
```
س: "على ماذا يجب أن أركز في اجتماع أعمال في الولايات المتحدة؟"
→ تفعيل الأداة US_BIZ_001
→ دليل التكيف الثقافي وإدارة الوقت وأسلوب التواصل
```

### المثال 3: الامتثال التنظيمي
```
س: "ما هي الإجراءات التي يجب أن أتبعها لمعالجة بيانات العملاء في الاتحاد الأوروبي؟"
→ تفعيل الأداة SA_DATA_001 و GDPR ذات الصلة
→ الامتثال لـ GDPR وإدارة الموافقة وإجراءات نقل البيانات
```

## 🛠 كيفية المساهمة (Contributing)

### 1. اقترح أداة جديدة
```bash
# اقترح مجال جديد
moai contribute --domain "القانون التجاري السعودي" --category "legal"
```

### 2. تحسين الأداة الموجودة
```bash
# تحسين بناءً على التعليقات
moai improve DOMAIN_ID --feedback "محتوى إضافي مطلوب"
```

### 3. إضافة توطين جديد
```bash
# إضافة توطين دولة جديدة
moai add-locale --country "الإمارات" --code "ar_AE"
```

### 4. تحسين التوثيق
- تحرير ملفات التوطين في `/skills/moai/references/locale/`
- تقديم Pull Request على GitHub
- تحديث أدلة التكيف الثقافي
