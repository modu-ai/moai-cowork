# 소셜 미디어 (Social Media)

> moai-content v1.0.0

## 개요

7개 주요 소셜미디어 플랫폼에 최적화된 콘텐츠를 제작한다.
2026년 인스타그램의 프라이빗 소셜 강화, 릴스 20분 확장, 관심사 기반 알고리즘 전환, 쿠팡·네이버 스토어 연동 커머스 트렌드를 반영한다.
사용자 행동 패턴 (활동 시간, 압축소비 트렌드, 신뢰 기반 마케팅)을 중심으로 설계한다.

⚠️ **AI 생성 콘텐츠 주의**: AI가 생성한 소셜미디어 콘텐츠는 사실 확인 후 게시하세요. 수치, 통계, 인물 정보는 반드시 원출처를 검증하세요.

## 트리거 키워드

소셜 미디어, 인스타그램, 릴스, 유튜브 쇼츠, 틱톡, 링크드인, 스레드, X, 트위터, 카카오채널, 네이버 클립, SNS 콘텐츠, 해시태그, 숏폼

## 지원 플랫폼 규격 (7개)

### 1. 인스타그램 (Instagram)

**2026년 알고리즘 핵심 변화:**
- **프라이빗 소셜 강화**: DM·공유·전송이 좋아요보다 5~10배 강한 신호
- **릴스 최우선 + 롱폼 확장**: 릴스 최대 20분, 기존 피드 광고 대비 30% 높은 참여율
- **관심사 기반 전환**: 팔로워 수 → 주제 일관성이 알고리즘 결정
- **검색 엔진화**: 해시태그 무력화, 텍스트·이미지 내 사물·ALT 텍스트 분석
- **커머스 연동**: 쿠팡·네이버 스토어 연동 광고 부상

**핵심 성과 지표 (2026):**
- DM 발송, 저장(Save), 공유(Share) — 최우선 지표
- 시청 시간(Watch Time) — 릴스 핵심
- 댓글 > 좋아요 순

**콘텐츠 규격:**
| 포맷 | 비율 | 최적 길이 | 특징 |
|------|------|-----------|------|
| 릴스 | 9:16 | 15~60초 (정보성 3~10분도 가능) | 최우선 노출 |
| 피드 사진 | 1:1 / 4:5 | — | 브랜드 아이덴티티 |
| 카루셀 | 1:1 | 5~10장 | 저장율 높음 |
| 스토리 | 9:16 | 15초 | DM 유도, 친밀감 |

**인스타 최적화:**
- 활동 피크: **평일 19~24시, 목요일 22시 (DM 최다)**
- 토요일 오전 10시~오후 2시
- 해시태그: 3~5개 이내 + 대체 텍스트(ALT) 활용
- CTA: "좋아요"보다 "이 정보가 필요한 친구에게 공유해보세요"

---

### 2. 스레드 (Threads)

**특징 및 전략:**
- X(트위터) 대체 포지셔닝, 인스타그램과 계정 연동
- 텍스트 기반 대화·토론 문화
- 팔로워 없어도 외부 공개 가능 (Fediverse 지원)

**콘텐츠 규격:**
- 텍스트: 최대 500자
- 이미지: 최대 10장
- 영상: 최대 5분

**콘텐츠 전략:**
- 의견·관점 공유 → 토론 유도
- 짧고 날카로운 인사이트 (50~100자)
- 인스타 릴스와 크로스 포스팅

---

### 3. X (구 Twitter)

**특징 및 전략:**
- 실시간성, 트렌딩 화제, B2B·테크 커뮤니티 강세
- 유료 구독(X Premium)으로 긴 글·영상 포스팅 가능

**콘텐츠 규격:**
- 무료 계정: 140자
- X Premium: 25,000자
- 영상: 최대 2분 20초 (무료), 더 길게 (Premium)

**콘텐츠 전략:**
- 스레드(Thread) 형식으로 긴 내용 분할
- 실시간 이슈 반응 (반응 속도가 핵심)
- 링크 포함 시 노출 제한 → 댓글에 링크 첨부 전략

---

### 4. 링크드인 (LinkedIn)

**특징 및 전략:**
- B2B·전문가 네트워크 최강, 의사결정자 직접 접근 가능
- 2026년 사용자 급증 중

**콘텐츠 규격:**
| 포맷 | 특징 |
|------|------|
| 텍스트 포스트 | 최대 3,000자, 단락 구분 중요 |
| 문서/PDF | 카루셀 형식, 다운로드 수 높음 |
| 영상 | 최대 10분, 자막 필수 |
| 아티클 | SEO 최적화, 긴 형식 전문 콘텐츠 |

**링크드인 전략:**
- 개인 브랜딩 중심: 창업자·임원의 개인 계정 활성화
- 산업 인사이트 + 개인 경험 결합
- 해시태그 3~5개 (관련 업종 태그)
- CTA: 댓글 질문 유도 (알고리즘 상 댓글 = 강력한 신호)

---

### 5. 유튜브 쇼츠 (YouTube Shorts)

**특징 및 전략:**
- 유튜브 알고리즘과 연동, 장편 유튜브로의 유입 통로
- 구글 검색 결과에 쇼츠 직접 노출 (SEO 효과)

**콘텐츠 규격:**
- 비율: 9:16 (세로)
- 길이: 최대 60초
- 음악: 유튜브 라이브러리 또는 저작권 없는 음원

**콘텐츠 전략:**
- 처음 3초: 핵심 후킹 문구 (시청 지속률 결정)
- 자막: 자막 필수 (무음 시청 비율 높음)
- 시리즈화: 관련 쇼츠 플레이리스트로 시청 시간 증대
- 장편 영상의 하이라이트 클립 재활용

---

### 6. 카카오 (채널 + 오픈채팅)

**카카오채널:**
- 국내 최대 모바일 메신저 기반, 국내 고객 직접 소통 최적
- 친구 추가 기반 알림 발송 (높은 오픈율)

**콘텐츠 규격:**
| 타입 | 특징 |
|------|------|
| 텍스트 메시지 | 최대 1,000자, 자주 사용 |
| 이미지 메시지 | 800x400px 권장 |
| 카드형 메시지 | 버튼 포함, CTA 최적 |
| 쿠폰 | 이커머스·오프라인 매장 필수 |

**전략:**
- 발송 시간: 오전 10~11시, 점심 12~13시, 저녁 19~21시
- 주 1~2회 발송 (과다 발송 시 차단율 증가)
- 채널 전용 혜택 제공 (쿠폰, 사전 정보 등)

---

### 7. 네이버 클립 (Naver Clip)

**특징 및 전략:**
- 네이버 검색과 직접 연동 (네이버 SEO 효과)
- 2025~2026년 C-Rank 알고리즘에서 영상 콘텐츠 우대 시작

**콘텐츠 규격:**
- 비율: 9:16
- 길이: 최대 60초 (숏폼)
- 연동: 네이버 블로그·스마트플레이스 연계

## 워크플로우

### 1단계: 목적 및 플랫폼 선택
- 목표: 브랜드 인지 / 팔로워 성장 / 리드 수집 / 판매 전환
- 대상 플랫폼 선택 (최대 3개 권장 — 집중력 유지)

### 2단계: 콘텐츠 캘린더 설계
```
주간 콘텐츠 계획 예시:

월: 교육 콘텐츠 (How-to, 인사이트)
화: 제품·서비스 소개 (소프트 세일)
수: 고객 사례·UGC 공유
목: 업계 뉴스·트렌드 (참여 유도)
금: 비하인드·팀 스토리 (신뢰 구축)
토: 라이브·Q&A
일: 베스트 콘텐츠 재활용
```

### 3단계: 콘텐츠 제작

**인스타그램 릴스 스크립트 구조:**
```
0~3초: 후킹 (Hook) — "XX 모르면 손해, 지금 바로 확인하세요"
3~10초: 문제 제시 — 독자의 고통 공감
10~45초: 해결책 — 핵심 내용 (3가지 이내)
45~60초: CTA — "저장하고 나중에 다시 보세요"
```

**링크드인 포스트 구조:**
```
첫 줄: 스크롤 멈추는 첫 문장 (질문 또는 충격적 통계)
...더 보기 클릭 유도

본문: 3~5개 단락 (각 2~3문장)
- 개인 경험 + 인사이트
- 구체적 수치

결론: 배운 점 1가지

CTA: 의견을 댓글로 남겨주세요 / 공감하시면 리포스트
```

### 4단계: 해시태그 전략

**플랫폼별 권장 해시태그 수:**
| 플랫폼 | 권장 수 | 전략 |
|--------|---------|------|
| 인스타그램 | 3~5개 | 2026년 대형 태그 효과 급감 |
| X | 1~2개 | 트렌딩 태그만 |
| 링크드인 | 3~5개 | 업종 특화 태그 |
| 유튜브 쇼츠 | 3개 | 제목·설명에 키워드 포함 |
| 카카오 | 불필요 | |
| 네이버 클립 | 5~10개 | 검색 연동 중요 |

### 5단계: 성과 분석 및 최적화
- 주간 지표 리뷰: 도달, 참여율, 팔로워 증감, 전환
- A/B 테스트: 게시 시간, 후킹 문구, 포맷 비교
- 베스트 콘텐츠 재활용 (Repurposing)

## 참고 사항

### 2026년 소셜미디어 핵심 인사이트
- **압축소비 트렌드**: 긴 탐색 없이 AI·SNS가 먼저 제안 → 첫 3초 후킹 필수
- **신뢰 마케팅**: "마케팅처럼 보이지 않는 마케팅"이 2026년 핵심
- **커머스 연동**: 인스타 → 쿠팡/네이버 스토어 직결 전환 전략 부상
- **크리에이터 이코노미**: 팔로워 수보다 전환율과 관계 깊이가 중요

### 소셜미디어 사용 패턴
- 인스타 DM 최다: 목요일 22시
- 전반적 피크: 평일 19~24시
- 주말: 토요일 오전 활동 높음
- Z세대: 숏폼 (릴스, 쇼츠, 클립) 70% 이상 소비
- 밀레니얼: 카카오톡 채널 구독률 높음

## 문제 해결

- **플랫폼 규격 변경**: 소셜미디어 플랫폼은 알고리즘과 규격을 수시로 변경합니다. 최신 정보는 각 플랫폼 공식 업데이트를 확인하세요.
- **브랜드 가이드 미제공**: 브랜드 컬러, 톤앤매너, 금지어가 있다면 미리 알려주세요. 없으면 일반 친화적 문체로 작성합니다.
- **이미지 생성 실패**: 이 스킬은 텍스트 콘텐츠를 생성합니다. 비주얼 슬라이드는 `card-news` 스킬을, 영상 스크립트는 `media-production` 스킬을 활용하세요.
- **저작권·초상권 이슈**: AI가 생성한 콘텐츠에 특정 인물·브랜드가 언급된 경우 게시 전 법적 검토가 필요할 수 있습니다.

## 이 스킬을 사용하지 말아야 할 때

- **블로그 포스팅**: 장문 SEO 블로그 글 작성은 `blog` 스킬이 더 적합합니다.
- **카드뉴스·슬라이드**: 비주얼 중심 캐러셀 콘텐츠는 `card-news` 스킬을 사용하세요.
- **이메일 뉴스레터**: 구독자 대상 뉴스레터는 `newsletter` 스킬을 사용하세요.
- **광고 카피·헤드라인**: 전환 최적화 마케팅 문구는 `copywriting` 스킬이 더 적합합니다.
- **유튜브 영상 기획**: 유튜브 채널 전략, 영상 편집 가이드는 `media-production` 스킬을 사용하세요.
