# moai-content

크리에이티브 콘텐츠 플러그인 — 카드뉴스, 랜딩페이지, 뉴스레터, 카피라이팅, 블로그, 소셜미디어, 미디어 프로덕션.

## 스킬

| 스킬 | 설명 | 상태 |
|------|------|:----:|
| [card-news](./skills/card-news/) | 인스타 카드뉴스 — AI 이미지 생성 기반 캐러셀 제작. 잡지 SOP, AI 글쓰기 방지 기법 적용 | ✅ 완성 |
| [landing-page](./skills/landing-page/) | 랜딩 페이지 빌더 — 고전환율 원페이지 설계, CTA 최적화, 디자인 원칙 적용 | ✅ 완성 |
| [copywriting](./skills/copywriting/) | 카피라이팅 — 마케팅 카피, 헤드라인, CTA, 광고 캠페인, 비주얼 스토리텔링 | ✅ 완성 |
| [newsletter](./skills/newsletter/) | 뉴스레터 엔진 — 기획~발행 전 과정, 구독자 확보 전략, 콘텐츠 캘린더 연동 | ✅ 완성 |
| [media-production](./skills/media-production/) | 미디어 프로덕션 — Remotion 영상, 유튜브 채널 기획, 팟캐스트, 전자책 출판 | ✅ 완성 |
| [blog](./skills/blog/) | 블로그 콘텐츠 — 네이버 블로그, 티스토리, 브런치, 미디엄, 워드프레스 최적화 포스팅 | 📝 초안 |
| [social-media](./skills/social-media/) | 소셜 미디어 콘텐츠 — 인스타그램, 유튜브 쇼츠, 틱톡, 트위터/X, 링크드인 | 📝 초안 |

```
"AI로 돈 버는 법" 주제로 인스타 카드뉴스 10장 만들어줘
```

```
유튜브 채널 기획서 써줘. 개발자 취업 정보 채널, 구독자 10만 목표.
```
